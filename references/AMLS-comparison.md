# 2026 台北 EMTP 複訓 vs AMLS 4th Edition — 對標比較

> 用途：講師備課參考，了解本課程在國際 EMS 教育中的定位
> 建立日期：2026-02-26
> 資料來源：NAEMT AMLS 官方課程資訊、AMLS 4th Edition (2024) 章節結構、NREMT/NCCP 文件

---

## 一、課程基本資料對比

| 面向 | 本課程（台北 EMTP 複訓） | AMLS 4th Edition (2024) |
|------|----------------------|------------------------|
| 主辦 | 台北市 EMS 系統 | NAEMT / NAEMSP |
| 時數 | **4 小時**（單次模組） | **16 小時**（2 天）/ Hybrid 8hr 線上 + 1 天面授 |
| 複訓 | 依台灣法規 | 4 年效期，8 小時 refresher |
| 涵蓋範圍 | ALS 病生理 + PK/PD（聚焦 shock + cardiac arrest） | 14 章全面內科急症（呼吸、心血管、休克、敗血、神經、毒物、內分泌...） |
| 教學理念 | 病生理決策導向（「不背流程，理解為什麼」） | 病理生理 + 鑑別診斷 + 臨床推理（Assessment Pathway） |
| 情境案例 | 4 案（張力性氣胸、低血容性休克、過敏性休克、OHCA-VF） | 80 個全新模擬情境（4th Ed） |
| 評量 | Pre 10 題 + Post 20 題 + Mini OSCE | Pre-test + Post-test ~50 題（通過 76%）+ 情境站 |
| 學分認可 | 台北市衛生局 | 16 hr CAPCE / NREMT 認可 |

## 二、教學理念比較

### 2.1 共同理念

兩者都反對「背演算法」，強調 **pathophysiology-driven clinical reasoning**：

| 共同點 | 本課程 | AMLS |
|--------|--------|------|
| 病生理為決策基礎 | 每個處置回到「壞掉的是哪個生理環節」 | Assessment Pathway 要求辨識 cardinal presentation 的生理原因 |
| Case-based learning | 4 個漸進式情境 + debrief | 80 個情境 + 小組討論 + debriefing |
| 反對死背 protocol | 明確寫在教學目標 | 與 ACLS algorithm-driven 做出區隔 |
| 休克早期辨識 | 強調代償期（BP 還正常時就要發現） | 強調 shock 初期可逆性 |

### 2.2 關鍵差異

| 面向 | 本課程 | AMLS |
|------|--------|------|
| **核心框架** | **「四柱」**：Oxygenation → Ventilation → Perfusion → Metabolism | **Assessment Pathway**（5 步驟動態循環） |
| **推理方向** | **演繹法**（框架 → 案例）：先教框架，再用案例驗證 | **歸納法**（案例 → 框架）：從案例呈現中建構鑑別診斷 |
| **藥理深度** | ★★★★★ 獨立模組，含 PK/PD、途徑選擇、特殊情境調整 | ★★★ Ch.2 臨床藥理概論，較少 PK/PD 機轉 |
| **鑑別診斷** | 窄而深（聚焦 4 種休克 + cardiac arrest） | 廣而全（14 章涵蓋所有內科急症 DDx） |
| **團隊動力** | 未涵蓋（討論制，非模擬） | 部分涵蓋（CRM 元素融入情境） |

## 三、核心框架逐項對比

### 3.1 系統性評估框架

**本課程：「生理四柱」+ 灌流三角**

```
Oxygenation (SpO2) → Ventilation (EtCO2) → Perfusion (BP/HR/CRT) → Metabolism (pH/Lactate)
                                                    |
                                            Pump - Pipe - Tank
                                          (心臟)  (血管)  (容量)
```

**AMLS：Assessment Pathway（5 步驟動態循環）**

```
Step 1: Initial Observations (Scene + Cardinal presentation + Primary survey)
    |
Step 2: First Impression (Sick/Not Sick → 初步 DDx → 立即處置)
    |
Step 3: Detailed Assessment (OPQRST + SAMPLER + Secondary survey + Diagnostics)
    |
Step 4: Refine DDx (Clinical reasoning → Life-threatening / Critical / Non-emergent)
    |
Step 5: Ongoing Management (Reassess → Refine → Modify → Loop back)
```

**比較分析**：
- 本課程的框架更**機轉導向**（直接指向「哪個系統壞了」）
- AMLS 的 Pathway 更**流程導向**（引導臨床推理的思考順序）
- 兩者互補：本課程的框架可以嵌入 AMLS Step 2-4 作為「判斷壞掉的系統」的工具

### 3.2 休克分類與教學

| 面向 | 本課程 | AMLS (Ch.5 Shock) |
|------|--------|-------------------|
| **分類法** | 過程導向：代償 → 失代償 → 不可逆（三階段） | 類型導向：Hypovolemic / Cardiogenic / Distributive / Obstructive |
| **早期辨識** | ★★★★★ 明確強調「BP 正常 ≠ 沒有休克」，列出代償期徵兆 | ★★★★ 強調初期可逆，但較少強調代償期 vital sign 陷阱 |
| **Pump-Pipe-Tank** | 核心教學框架 | 概念有提及但非主要框架 |
| **機轉連結** | 每種 shock 回到「哪個環節壞掉」 | 每類 shock 有病理生理說明 |

**互補點**：本課程的「三階段」分類可以疊加在 AMLS 的「四類型」分類上 — 先辨識類型，再判斷階段。

### 3.3 藥理學

| 面向 | 本課程 M02 | AMLS Ch.2 (4th Ed) |
|------|-----------|-------------------|
| **定位** | 獨立核心模組（佔課程 50%） | 基礎章節（14 章之一） |
| **PK/PD 深度** | ★★★★★ ADME 四步驟 x 特殊情境（休克/心停/老人/低體溫） | ★★ 臨床藥理概論，較少機轉層面 |
| **途徑選擇** | IV/IO/IM/IN 完整決策矩陣 + 休克禁用 IM | 有提及途徑，無系統性決策框架 |
| **情境調整** | 休克減量 50%、心停蓄積效應、ROSC 後風險 | 部分提及，較少量化建議 |
| **藥物短缺** | 未涵蓋 | 4th Ed 新增藥物短缺管理 |
| **速查表** | 含 push-dose epi 配製、drip rate 計算 | 劑量表散在各章 |

**這是本課程最大的差異化優勢。** AMLS 的藥理是「夠用」等級；本課程提供了**可操作的 PK/PD 決策框架**，這在國際 EMS 教育中相當罕見。

## 四、情境案例設計對比

### 4.1 設計哲學

| 面向 | 本課程 | AMLS |
|------|--------|------|
| 案例數量 | 4 個（精選） | 80 個（全面） |
| 進程設計 | 時間軸漸進（T0 → T+3 → T+5 → T+10），呈現藥物 onset/effect | 情境推進但較少精確時間軸 |
| 配對邏輯 | S01-S02（通氣 vs 灌流）、S03-S04（分布性 vs 心停）對比配對 | 按疾病系統分組 |
| Debrief | 「壞掉的是什麼？修的是什麼？失敗怎麼調？」三問框架 | 引導式反思，較開放 |
| 藥物整合 | 每案連結 PK/PD（為何選此途徑、onset 預期、蓄積風險） | 藥物作為治療選項呈現，較少 PK/PD 連結 |

### 4.2 主題交叉對比

| 本課程情境 | AMLS 對應章節 | 深度比較 |
|-----------|-------------|---------|
| **S01 張力性氣胸**（阻塞性休克） | Ch.3 Respiratory + Ch.5 Shock | 本課程聚焦「呼吸→循環連鎖」；AMLS 更全面但此主題深度相當 |
| **S02 低血容性休克**（代償崩潰） | Ch.5 Shock | 本課程的三階段進程模型 > AMLS 的分類說明 |
| **S03 過敏性休克**（Epi 藥理） | Ch.5 Shock + Ch.2 Pharmacology | **本課程在 Epi PK/PD 上明顯更深**（三重作用機轉 + 途徑選擇 + biphasic 風險） |
| **S04 OHCA-VF**（終點路徑） | Ch.4 Cardiovascular | ACLS 更直接對應；AMLS 較少深入 arrest pharmacology |

## 五、評量系統對比

| 面向 | 本課程 | AMLS |
|------|--------|------|
| Pre-test | 10 題 MCQ | 有（題數不固定） |
| Post-test | 20 題 MCQ | ~50 題 MCQ |
| 通過門檻 | **80%** | **76%** |
| 認知層級分布 | 記憶/理解 30% + **應用 50%** + 臨床判斷 20% | 混合（依版本/講師） |
| 技能評量 | Mini OSCE（5 分鐘口頭抽卡） | 情境站（scenario-based） |
| 補考機制 | 1 週內 targeted remediation（10 題 + 1 案口述） | 依訓練中心規定 |

本課程的通過門檻（80%）比 AMLS（76%）更高，且認知層級偏重「應用」而非「記憶」。

## 六、本課程獨特優勢（AMLS 未涵蓋或較弱的部分）

### 6.1 PK/PD 實戰決策框架
- **休克禁用 IM**（藥物蓄積在肌肉，復甦後 late dump）
- **IO 肱骨 > 脛骨**（更快到達中央循環）
- **心停蓄積效應**（多次 Epi → ROSC 後血壓暴升/心律不整）
- **休克減量原則**（Sedation 減 50%）
- AMLS 4th Ed Ch.2 有藥理概論，但缺乏這種「情境 x 途徑 x 劑量」的決策矩陣

### 6.2 EtCO2 酸鹼整合
- 教學員「不要把代謝性酸中毒患者的 EtCO2 通氣回正常值」（會破壞呼吸代償）
- AMLS 較少在 EtCO2 與酸鹼平衡之間做這種整合教學

### 6.3 「三問」Debrief 框架
1. 現在壞掉的是哪個生理環節？
2. 你選的處置在修正哪個環節？
3. 若處置失敗，下一步怎麼調整？

比 AMLS 的開放式 debrief 更結構化，且可跨情境重複使用。

### 6.4 情境時間軸設計
- 每案有精確時間軸（T0 → T+3 → T+5 → T+10），學員看到藥物 onset 的真實時間感
- AMLS 情境有進程推進，但較少精確到分鐘的時間軸

## 七、AMLS 可借鏡之處（本課程未來可強化的方向）

### 7.1 AMLS Assessment Pathway
- 非線性動態評估框架，可隨時回到任何步驟
- 建議：可將本課程的「四柱框架」嵌入 AMLS Pathway Step 2-4，形成整合

### 7.2 廣度涵蓋
- AMLS 14 章涵蓋：敗血症、神經急症、內分泌/代謝（DKA/HHS）、毒物、精神衛生、婦科
- 本課程聚焦 shock + arrest，若未來擴展可參考 AMLS 章節架構

### 7.3 鑑別診斷訓練
- AMLS 強調從 cardinal presentation 出發，列出 DDx 並逐步排除
- 本課程的情境已知診斷，較少訓練「不確定診斷時的推理」

### 7.4 藥物短缺管理（4th Ed 新增）
- 實務面：當常用藥缺貨時的替代方案

### 7.5 團隊動力（CRM）
- AMLS 情境融入 Crisis Resource Management 元素
- 可考慮加入團隊溝通 / 閉環溝通的教學

## 八、定位總結

```
                        +-------------------------------------------+
                        |           AMLS 4th Ed (16 hr)             |
                        |                                           |
                        |  Ch1  Assessment Pathway                  |
                        |  Ch2  Clinical Pharmacology               |
                        |  Ch3  Respiratory        +---------------------+
                        |  Ch4  Cardiovascular     | 本課程 (4 hr)        |
                        |  Ch5  Shock <------------| M01 病生理學         |
                        |  Ch6  Sepsis             | M02 PK/PD *****    |
                        |  Ch7  Infectious         | S01-S04 情境案例     |
                        |  Ch8  Neurologic         | Pre/Post + Mini OSCE|
                        |  Ch9  Mental Health      +---------------------+
                        |  Ch10 Abdominal                           |
                        |  Ch11 Endocrine/Metabolic                 |
                        |  Ch12 Women's Health                      |
                        |  Ch13 Environmental                       |
                        |  Ch14 Toxicology                          |
                        +-------------------------------------------+

重疊區域：休克 (Ch5) + 心血管 (Ch4) + 藥理 (Ch2)
本課程獨有：PK/PD 情境決策框架、給藥途徑矩陣、EtCO2 酸鹼整合
AMLS 獨有：廣泛內科 DDx、Assessment Pathway、CRM、藥物短缺
```

**一句話定位**：本課程是 AMLS Ch.2 + Ch.4 + Ch.5 的**精華深潛版** — 在藥理實戰應用（PK/PD）上的深度超越 AMLS，但涵蓋的疾病系統範圍遠窄於 AMLS 的 14 章全貌。
