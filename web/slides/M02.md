# M02 藥物動力學/藥效學
## PK/PD：掌握「時間」與「情境」

**2026 台北市 EMTP 複訓**

<div style="margin-top: 1.5rem; opacity: 0.85; font-size: 0.95em;">
在 EMS 現場，沒有時間等藥效，也沒有機會調整劑量。<br>
精準掌握 <strong>Onset</strong> 與 <strong>Duration</strong>，是 EMTP 的核心能力。
</div>

---

## 為什麼需要 PK/PD？

> 打進去 ≠ 馬上有效

<div class="alert-purple">
<strong>三個關鍵問題</strong><br>
① <code>Epinephrine 1mg IV push</code> → 幾秒後血壓上升？（Onset）<br>
② <code>Midazolam 5mg IM</code> → 幾分鐘後鎮靜？（Onset）<br>
③ RSI 前，Sedative/Paralytic 還在作用嗎？（Duration）
</div>

**藥物的時間軸就是生命線**

- 選錯途徑 → 起效慢 → 誤判 → 追加 → **過量**
- 不知 Duration → 插管視窗已關 → 危機
- 休克病人 → 全身 PK 改變 → 劑量要重算

## PK 四大步驟臨床意義

**ADME = Absorption → Distribution → Metabolism → Excretion**

### A — 吸收 (Absorption)

| 途徑 | 生物利用率 |
|------|-----------|
| **IV / IO** | **100%** — 零時差 |
| IM | 依賴肌肉血流，休克時**極差** |
| IN | 黏膜快，但劑量受限 |

::right::

### D — 分布 (Distribution)

休克病人 → **Centralization**（集中到心腦）
→ 周邊組織藥物不足，但心腦**相對過量**

<div class="alert-red">
⚠️ 休克病人給鎮靜藥：<br>
Start Low, Go Slow
</div>

### M + E — 代謝與排泄

- 老人 / 洗腎 → 代謝慢 → Duration 延長
- **低體溫** → 酵素活性下降 → 藥物超慢代謝
  > ROSC 後別急著追加鎮靜！

---

## 給藥途徑四路比較

| 途徑 | Onset | 適用 | 休克/瀕死 注意 |
|------|-------|------|----------------|
| **IV（上肢）** | 10–30 sec | ALS 黃金標準 | 推藥後 Flush；上肢 > 下肢 |
| **IO（肱骨）** | 30–60 sec | 休克/OHCA 首選 | 速度接近 Central Line |
| **IO（脛骨）** | 30–60 sec | 休克/OHCA | **需加壓輸液**才能進入循環 |
| **IM** | 5–15 min | 躁動、無 IV | **休克禁用！** Late Dump 危險 |
| **IN** | 3–5 min | 兒童、疼痛 | 每孔 < 1mL；鼻塞無效 |

<div class="alert-red">
🚫 <strong>休克病人 IM → 藥物積在肌肉</strong><br>
休克改善後集體釋放（Late Dump）→ 突發過量
</div>

---

## IO 細節：肱骨 vs 脛骨

<div class="alert-blue">
<strong>Humeral IO（肱骨）</strong><br>
→ 到達心臟速度接近 Central Line<br>
→ 休克/OHCA 首選上肢 IO
</div>

<div class="alert-amber">
<strong>Tibial IO（脛骨）</strong><br>
→ <strong>需加壓輸液（Pressure Bag / Push）</strong>才能進入循環<br>
→ 清醒病人注意：痛！<br>
→ 清醒使用前：Lidocaine 40mg IO 預先給藥
</div>

**IO 後務必：推藥後 Flush 20mL + 抬高肢體**

---

## 休克病人的 PK 調整

### 問題：CO ↓ → 肝腎血流 ↓

| 藥物類別 | 調整策略 |
|---------|---------|
| Sedation / Analgesia | **減量**，給藥速度放慢 |
| Paralytics | Onset 可能**延遲**（不要以為沒效補藥）|
| Pressors | 反應可能**劇烈**（受體已被內源 catecholamine 活化）|

<div class="alert-amber">
<strong>Distribution Centralization</strong><br>
→ 藥物集中衝擊心腦<br>
→ 原本安全的劑量 = 相對過量
</div>

---

## 心搏停止的 PK

**問題：完全無循環，全靠 CPR 推動血流**

<div class="alert-blue">
<strong>藥物到達心臟時間延長</strong><br>
→ IV/IO 後務必：<strong>Flush 20mL + 抬高肢體</strong>
</div>

### Epinephrine 累積陷阱

每 3–5 min 一次 Epi → **ROSC 時體內可能積累 3–5mg**

<div class="alert-red">
ROSC 後立即出現：<br>
• Rebound Hypertension<br>
• Rebound Tachycardia / VT/VF<br>
→ 這不是新問題，是 Epi 蓄積效應
</div>

## Epinephrine — 三種濃度陷阱

<div class="alert-red">
<strong>🚨 最常見致命用藥錯誤</strong><br>
1:1,000 vs 1:10,000 搞混
</div>

| 濃度 | 換算 | 用途 |
|------|------|------|
| **1:1,000** | 1mg / 1mL | IM（過敏性休克）|
| **1:10,000** | 1mg / 10mL | IV/IO（心停）|

::right::

### 各情境劑量

**Cardiac Arrest（心停）**
→ IV/IO 1mg（1:10,000）q 3–5 min

**Anaphylaxis（過敏）**
→ IM 0.3–0.5mg（**1:1,000 原液**）q 5–15 min

**Push-dose（升壓）**
→ 10–20 mcg IV q 1–2 min
→ 泡製：1mL(1:10,000) + 9mL NS = **10 mcg/mL**

---

## Push-dose Epinephrine

**現代 EMS 處理休克的關鍵技能**

<div class="alert-green">
<strong>泡製步驟</strong><br>
取 1mL（1:10,000）+ 9mL NS<br>
= 10 mcg / mL<br>
每次給 1–2 mL（10–20 mcg），每 1–2 分鐘 titrate 至 SBP > 90
</div>

### ⚠️ PseudoPEA 特殊情境

<div class="alert-red">
<strong>心臟還在跳（摸不到脈）→ 用 Push-dose（10–20 mcg）</strong><br>
<strong>絕對不能用 ACLS 劑量 1mg</strong><br>
→ 心臟仍有收縮力，全量 Epi 會直接打成 VF/VT
</div>

---

## Amiodarone

| 情境 | 劑量 |
|------|------|
| OHCA VF/pVT — 1st dose | **300 mg IV/IO bolus** |
| OHCA VF/pVT — 2nd dose | 150 mg IV/IO bolus |
| 非 arrest VT（活人）| **150 mg IV over 10–15 min** |

<div class="alert-red">
🚨 <strong>活人推太快 → Hypotension</strong><br>
死人才 Push；活人要 Drip（10–15 分鐘滴注）
</div>

**PK：IV Onset 數分鐘，脂肪蓄積久（超長半衰期）**

---

## Adenosine

**適應症：Stable SVT（穩定的上心室頻脈）**

<div class="alert-purple">
<strong>半衰期 < 10 秒</strong><br>
→ 必須在最靠近心臟的 IV<br>
→ Rapid push（快推）+ 20 mL flush<br>
→ 打太慢 = 沒效
</div>

| 劑量 | 用法 |
|------|------|
| 1st dose | 6 mg rapid IV push + 20mL flush |
| 2nd dose | 12 mg rapid IV push + 20mL flush |
| 3rd dose | 12 mg rapid IV push + 20mL flush |

<div class="alert-red">
🚫 <strong>禁用於 Irregular Wide-complex Tachycardia</strong>（可能是 AF + WPW）
</div>

## Midazolam（Dormicum）

**鎮靜 / 止痙**

| 途徑 | 劑量 |
|------|------|
| IV | 2–5 mg |
| IM | 5–10 mg |
| IN | 0.2 mg/kg（兒童，每孔 < 1mL）|

**PK：肝代謝，IV Onset 1–3 min**

<div class="alert-red">
🚨 <strong>呼吸抑制（Respiratory Depression）</strong><br>
→ 尤其併用 Opioid 時<br>
→ 老人/休克：<strong>劑量減半</strong>
</div>

::right::

## Fentanyl

**止痛**

| 途徑 | 劑量 |
|------|------|
| IV | 1–2 mcg/kg（通常 50–100 mcg）|
| IN | 1.5 mcg/kg（每孔 < 1mL）|

**PK：IV Onset 1–2 min，Duration 30–60 min**

不釋放組織胺 → 比 Morphine 更不影響血壓

<div class="alert-amber">
⚠️ <strong>Chest Wall Rigidity（胸壁僵硬）</strong><br>
極快推注大劑量時可能發生<br>
→ 無法 BVM！
</div>

---

## Nitroglycerin（NTG）— 肺水腫利器

**適應症：Warm + Wet（SCAPE / 急性肺水腫）**

| 途徑 | 劑量 | Onset |
|------|------|-------|
| Sublingual | 0.4 mg q 3–5 min | 1–3 min |
| SL 加倍（SCAPE）| **0.8 mg** | 1–3 min |
| IV drip | 10–200 mcg/min，titrate | < 1 min |

**作用機轉：**
- 低劑量 → 靜脈擴張 → 減前負荷
- 高劑量 → 動脈擴張 → 減後負荷

<div class="alert-red">
🚫 <strong>SBP < 90 禁用</strong><br>
🚫 <strong>右心梗（RV infarct）禁用</strong> — 右心靠前負荷維生，NTG 會讓它崩潰<br>
→ Inferior STEMI → 必查 RV lead (V4R) ST↑
</div>

---

## NTG 的 PK/PD 教學點

<div class="alert-amber">
<strong>SL NTG 在低灌流時吸收差！</strong><br>
→ 黏膜血流不足 → 吸收延遲<br>
= 跟 IM 在休克時失效是同樣道理
</div>

**低血壓時的 SL NTG：**
```
給藥 → 黏膜吸收極差 → 看起來沒效 → 追加
         ↓
血壓稍回升 → 黏膜血流恢復 → 積存的 NTG 全部吸收
         ↓
        血壓驟降！
```

**這就是「吸收 (Absorption) 依賴灌流」的臨床陷阱**

## 老年人 PK 調整

**問題：**
- 體脂增加 → 脂溶性藥物（Midazolam）分布廣 / 蓄積
- 肌肉減少 → IM 吸收更差
- 肝腎功能衰退 → 代謝/排泄慢

<div class="alert-amber">
<strong>Midazolam / Fentanyl：劑量減半</strong><br>
Start Low, Go Slow
</div>

::right::

## 升壓劑速查

**Norepinephrine**
- Septic/Distributive Shock：0.1–0.5 mcg/kg/min IV drip
- Cardiogenic Shock（Cold+Wet, SBP < 70）：首選升壓
- 泡製：4mg/250mL D5W = **16 mcg/mL**

<div class="alert-red">
⚠️ 外滲 → 組織壞死<br>
→ 優先 Central Line 或大口徑 IV
</div>

**Dobutamine（Cold+Wet, SBP 70–90）**
- 2–20 mcg/kg/min
- β1 為主 → CO↑；輕度 β2 → 後負荷↓
- 注意：增加心肌耗氧，AMI 時有風險

---

## Albuterol（支氣管痙攣）

**適應症：Bronchospasm（支氣管痙攣）**

```
2.5 mg nebulizer q 15–20 min PRN
```

**PK：吸入 Onset 5–15 min**

<div class="alert-amber">
⚠️ <strong>Side Effects</strong><br>
• Tachycardia（心跳加速）<br>
• Tremor（手抖）<br>
→ β2 selectivity 不是 100%
</div>

---

## 特殊情境速查

<div class="alert-red">
<strong>🩺 低體溫 (Hypothermia)</strong><br>
酵素活性下降 → 藥物代謝超慢<br>
→ ROSC 後<strong>別急著追加鎮靜</strong>
</div>

<div class="alert-purple">
<strong>🩺 PseudoPEA</strong><br>
心臟還在跳（超聲看得到） → Push-dose Epi 10–20 mcg<br>
→ 絕對不是 ACLS 劑量 1 mg
</div>

<div class="alert-blue">
<strong>🩺 OHCA Flush 原則</strong><br>
任何 IV/IO 給藥後：<strong>Flush 20mL + 抬高肢體</strong><br>
（沒有循環，要「推」藥進去）
</div>

## 藥物計算：Dopamine / NE Drip

**Rule of 6（傳統快算法）**

```
想要的劑量 (mcg/kg/min) × 體重 (kg) × 60 min
─────────────────────────────────────────────
           藥物濃度 (mcg/mL)
= 滴速 (mL/hr)
```

**Push-dose Epi 快速泡製**

```
1 mL（1:10,000，即 0.1mg/mL）
+ 9 mL NS
= 10 mL 共 100 mcg
= 10 mcg/mL
→ 每次 1–2 mL = 10–20 mcg
```

> 依照當地 protocol 使用規定配方與速率

## 模組總結 — 六大核心原則

<div class="alert-red">
① <strong>休克病人不要 IM</strong> — 打 IO/IV（Late Dump 危險）
</div>

<div class="alert-purple">
② <strong>Humeral IO 快於 Tibial IO</strong> — 休克/OHCA 首選肱骨
</div>

<div class="alert-amber">
③ <strong>老人鎮靜藥量減半</strong> — Start Low, Go Slow
</div>

<div class="alert-blue">
④ <strong>Push-dose Epi</strong> 是休克升壓利器（比單純輸液有效）
</div>

<div class="alert-green">
⑤ <strong>NTG 是肺水腫利器</strong>，但 SBP < 90 / 右心梗禁用
</div>

<div class="alert-red">
⑥ <strong>PseudoPEA 用 10–20 mcg Push-dose</strong>，不是 1 mg ACLS 劑量
</div>
