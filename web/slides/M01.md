## 核心觀念

### 生命徵象是代償的結果

<div class="alert-blue" style="margin-top: 1rem;">

**BP 80/40、HR 130** 不只是「休克數字」——
這是身體**拚命代償以維持灌流**的結果。

我們的介入（給水、給藥、插管）
必須**順應或矯正**這個代償過程，
不是只看數字治療。

</div>

::right::

<br><br>

### 瀕死生理四大支柱

| 支柱 | 指標 |
|------|------|
| **氧合** Oxygenation | SpO₂ |
| **通氣** Ventilation | EtCO₂ |
| **灌流** Perfusion | BP、Pulse、CRT、意識 |
| **代謝** Metabolism | pH、Base Excess |

---

## Oxygenation vs Ventilation

<div style="font-size: 0.9em;">

| | 氧合 Oxygenation | 通氣 Ventilation |
|---|---|---|
| **定義** | O₂ 從肺泡進入血液 | 空氣進出肺部（排 CO₂） |
| **主要指標** | **SpO₂**、PaO₂ | **EtCO₂**、RR、Tidal Volume |
| **常見問題** | 缺氧 Hypoxemia | 高碳酸 Hypercapnia |
| **生理機制** | V/Q mismatch、Shunt | Respiratory drive、肌力、氣道 |
| **處置邏輯** | **給氧** NRM / Nasal | **通氣支持** BVM / ETT |

</div>

## 氧氣解離曲線（ODC）

### 為什麼 SpO₂ 會「突然掉」？

<div class="alert-blue" style="margin-top: 0.5rem;">

S 型曲線的意義：

- **SpO₂ > 90%**（平坦段）：PaO₂ 大幅變化，SpO₂ 變化很小 → **看起來穩定**
- **SpO₂ 70–90%**（陡峭段）：PaO₂ 只要小幅下降，SpO₂ **急遽下掉** → **懸崖效應**

</div>

<div class="alert-red" style="margin-top: 0.5rem;">

⚠️ **臨床陷阱**

SpO₂ 從 99% 掉到 93% 你可能不緊張，但從 93% 到 85% 可能只要**幾秒鐘**。

**SpO₂ ≤ 93% = 已經站在懸崖邊！**

</div>

::right::

<br>

### 曲線右移 vs 左移

| | 右移（O₂ 更易釋放） | 左移（O₂ 更難釋放） |
|---|---|---|
| **pH** | ↓ 酸中毒 | ↑ 鹼中毒 |
| **CO₂** | ↑ 高碳酸 | ↓ 低碳酸 |
| **溫度** | ↑ 發燒 | ↓ 低溫 |
| **2,3-DPG** | ↑ | ↓ |

<div class="alert-amber" style="margin-top: 0.5rem; font-size: 0.85em;">

**記憶口訣**：組織缺氧時（酸、熱、高 CO₂）→ 曲線右移 → Hb 更願意「放手」讓 O₂ 給組織用 → 這是身體的代償機制。

</div>

## Oxy vs Vent：臨床決策點

### SpO₂ 掉 + RR **快**
→ Oxygenation 問題（肺炎、肺水腫）
→ **先給高濃度氧**

### SpO₂ 掉 + RR **慢/淺**
→ Ventilation 問題（OHCA、藥物過量、頭傷）
→ **單純給氧無效，需 BVM / 插管換氣**

### SpO₂ 正常但**意識改變**
→ 小心 **CO₂ retention**（Ventilatory failure）
→ 看 EtCO₂ 是否過高

::right::

<br><br>

<div class="alert-red">

⚠️ **最常見陷阱**

SpO₂ 低 → 直接插管

**錯！** 先確認是 Oxygenation 還是 Ventilation 問題。
SCAPE（Sympathetic Crashing Acute Pulmonary Edema，交感風暴性急性肺水腫）病人 NIV 比插管效果更好且更安全。

</div>

---

## 休克的動態進程

### 休克 ≠ 低血壓

> 休克是「**組織灌流不足**」
> 血壓下降是休克的**晚期徵兆**

<br>

| 階段 | 機制 | 徵象 | 處置 |
|------|------|------|------|
| **代償期** Compensated | 交感神經↑、Catecholamine surge | HR↑、SVR↑、BP 正常、焦慮 | **黃金期！早期介入** |
| **失代償期** Decompensated | 代償耗竭 → 無氧代謝 → 乳酸堆積 | **BP↓**、HR↑↑、意識不清、少尿 | 積極輸液 / 升壓 |
| **不可逆期** Irreversible | 多重器官衰竭、DIC | 即使 BP 拉回仍死亡 | — |

## 灌流評估三角

### Pump（心臟）
- Rate、Rhythm、Contractility
- 脈壓差窄（Narrow PP = SBP − DBP < 25 mmHg）→ 暗示心輸出量不足
- 心音低沉、末梢濕冷

### Pipe（血管）
- SVR（後負荷）
- 脈壓差、CRT、皮膚溫度

### Tank（容量）
- Preload（前負荷）
- **空**：JVD(-)、黏膜乾、肺音乾淨 → 輸液
- **滿**：JVD(+)、雙側囉音、下肢水腫 → **禁大量輸液**

::right::

<br>

<div class="alert-amber">

**⚠️ 陷阱**

Tank 空 vs Tank 滿 vs Pump 不力
三者**都可能低血壓**
但處置**完全不同**

</div>

---

## 鑑別決策樹：低血壓怎麼判？

<pre>
低血壓 (SBP < 90)
  │
  ├─ 皮膚冰冷 + JVD(-) + 肺音乾淨
  │     → Tank 空    → 輸液          (S02 低血容性)
  │
  ├─ 皮膚溫暖 + JVD(-) + ±wheeze
  │     → Pipe 鬆    → Epinephrine   (S03 過敏性)
  │
  ├─ 皮膚冰冷 + JVD(+) + 肺音乾淨
  │     → Tank 被擋  → 減壓          (S01 張力性氣胸)
  │
  └─ 皮膚冰冷 + JVD(±) + 雙側囉音
        → Pump 不力  → 升壓 / 強心   (S05 心因性)
</pre>

## 心因性休克

### Pump 壞了——心臟打不出去

<div class="alert-blue">

**Forward failure（前向衰竭）**
心輸出量↓ → 全身低灌流 → **Cold**

**Backward failure（後向衰竭）**
血液回堵 → 肺水腫 → **Wet**

**Cold + Wet** = 最難處理的急重症之一

</div>

### 常見成因
- AMI / 心肌炎（最常見）
- 高血壓危象 → SCAPE
- 快速心房顫動 / VT
- 心包填塞 / 大面積 PE

::right::

<br><br>

### 急性肺水腫病生理

<pre>
左心衰竭
  → 左心室舒張末壓↑
  → 肺靜脈壓↑
  → 肺微血管靜水壓↑
  → 血漿滲入肺間質/肺泡
  → SpO₂↓ + 呼吸困難
    + 雙側囉音 (crackles)
</pre>

---

## SCAPE

### Sympathetic Crashing Acute Pulmonary Edema

<br>

**機制**：交感風暴 → 血壓飆高 → 後負荷暴增 → 左心室打不過去 → 急性肺水腫

**特徵**：高血壓 + 呼吸窘迫 + 粉紅泡沫痰（**惡化極快：分鐘級**）

<br>

<div style="display: grid; grid-template-columns: 1fr 1fr; gap: 1rem;">

<div class="alert-blue">

**處置**

- **NTG**（減後負荷）
- **CPAP / BiPAP**（減回心血量 + 改善氧合）
- **坐姿**

</div>

<div class="alert-red">

**不要急著插管！**

NIV 往往在 **5–10 分鐘**內顯著改善
比插管侵入性低、效果更快

</div>

</div>

## Nohria-Stevenson 矩陣

<pre style="font-size: 1em; line-height: 1.6;">
                   灌流 (Perfusion)
                Warm（溫暖）   Cold（冰冷）
            ┌─────────────┬─────────────┐
Dry（乾）   │  A: 穩定     │  L: 低灌流   │
無肺水腫    │  觀察         │  輸液試探    │
            ├─────────────┼─────────────┤
Wet（濕）   │  B: 肺水腫   │  C: 心因性   │
有肺水腫    │  NTG + NIV   │  升壓 + 強心  │
            └─────────────┴─────────────┘
</pre>

---

## 四象限處置

| 象限 | 臨床表現 | 處置方向 |
|------|---------|---------|
| **A Warm+Dry** | 穩定代償，生命徵象正常 | 口服藥調整，不需急處置 |
| **B Warm+Wet** | 呼吸困難 + 囉音 + SpO₂↓，BP 正常或偏高 | **NTG + CPAP/BiPAP**（SCAPE 即屬此類） |
| **L Cold+Dry** | 低血壓 + 末梢冰冷，肺音乾淨 | 小量輸液試探（注意：可能右心衰或 PE） |
| **C Cold+Wet** | 低血壓 + 末梢冰冷 + 囉音 + SpO₂↓ | **最棘手**：不能輸液（更濕）、不能 NTG（更低）→ 升壓 + 強心 |

## PseudoPEA

<div class="alert-amber">

**PseudoPEA ≠ 心跳停了**

心臟還在跳，但 SBP < 40–50 mmHg
低到摸不到脈搏。

它不是 Cardiac Arrest，
它是 **Extreme Shock**。

</div>

### 為何重要

- 42–86% 的 PEA 可能是 PseudoPEA
- True PEA 存活率 < 1%
- PseudoPEA（有心臟收縮）ROSC 率可達 **70–94%**
- **POCUS 可作為重要輔助，但不可延長 rhythm-check 暫停**

::right::

<br>

### True PEA vs PseudoPEA

| 項目 | True PEA | PseudoPEA |
|------|----------|-----------|
| 心臟狀態 | 無有效收縮 | 有收縮但極弱 |
| 處置 | ACLS：CPR + Epi 1mg | **重度休克**：push-dose epi 10–20 mcg |
| Epi 劑量 | 1 mg | **10–20 mcg**（全量會打成 VF） |
| CPR | 持續壓胸 | **不要為鑑別延長暫停**；若快速確認有收縮，優先處理 low-flow shock |

---

## 休克 → 心跳停止的連續體

<pre style="font-size: 1.1em; line-height: 2;">
正常
  → 代償休克
    → 失代償休克
      → PseudoPEA   ← 最後的逆轉窗口
        → True PEA
          → Asystole
</pre>

<div class="alert-red" style="margin-top: 1.5rem;">

所有 S01–S05 的惡化終點都在這條線上。
**早期識別 PseudoPEA = 在不拖慢 resuscitation 的前提下，及早轉向 low-flow shock 思維。**

</div>

## 乳酸代謝：為什麼休克會酸？

### 正常 vs 缺氧時的能量代謝

<div class="alert-blue" style="margin-top: 0.5rem;">

**有氧代謝（正常）**

葡萄糖 → 丙酮酸 → 粒線體（TCA cycle）
→ **36 ATP** + CO₂ + H₂O

</div>

<div class="alert-red" style="margin-top: 0.5rem;">

**無氧代謝（缺氧/休克）**

葡萄糖 → 丙酮酸 → **乳酸 + H⁺**
→ 只產 **2 ATP**（效率剩 1/18）

</div>

<div class="alert-amber" style="margin-top: 0.5rem; font-size: 0.85em;">

**乳酸堆積的後果**：pH↓ → 心肌收縮力↓ → 血管對 catecholamine 反應↓ → 休克更惡化 → 產生更多乳酸 → **死亡螺旋**

</div>

::right::

<br>

### 院前判讀：用 EtCO₂ 間接看酸中毒

| EtCO₂ 型態 | 可能原因 |
|---|---|
| **低（< 20 mmHg）+ RR 快** | 代謝性酸中毒代償（乳酸/DKA） |
| **低 + 波形規律** | 嚴重低灌流 / 即將 arrest |
| **高（> 45 mmHg）** | 通氣不足 / CO₂ 蓄積 |
| **驟降** | 循環衰竭 / 心跳停止 |

<div class="alert-green" style="margin-top: 0.5rem; font-size: 0.85em;">

**臨床重點**：乳酸清除率（lactate clearance）是 ICU 評估復甦成效的金標準。院前雖無法直接測量，但**恢復灌流 = 清除乳酸的唯一途徑**。

EtCO₂ 回升（從極低值）= 灌流可能在改善。

</div>

## 酸鹼與代謝

### 院前沒有 ABG，但 EtCO₂ 是酸中毒的間接指標

<br>

**代謝性酸中毒（Metabolic Acidosis）**

- 原因：組織缺氧（乳酸）、DKA、腎衰竭
- 代償反應：Kussmaul breathing → RR↑ + TV↑ → EtCO₂↓

<br>

<div class="alert-amber">

**⚠️ EtCO₂ 低 ≠ 要補換氣**

休克/DKA 病人 EtCO₂ 偏低是**代償結果**
不要刻意 BVM 到正常值（35–45 mmHg）
否則破壞代償 → **酸中毒惡化**

</div>

## 常見藥物生理效應

<div style="font-size: 0.78em;">

| 藥物/處置 | 作用目標 | 預期效應 | 風險 |
|----------|---------|---------|------|
| **Fluids** | Tank（Preload）| SV↑ → CO↑ → BP↑ | Fluid overload。**Cold+Wet 禁大量！** |
| **Epinephrine** | α1（血管）+ β1（心臟）| SVR↑、Contractility↑ | 心肌耗氧↑、心律不整 |
| **Nitroglycerin** | 靜脈/動脈擴張 | 減前負荷 → 減肺水腫 | **SBP < 90 禁用。右心梗禁用。** |
| **Amiodarone** | K⁺/Na⁺/Ca²⁺ Channel | 延長動作電位、抑制異位放電 | **Hypotension**（推太快）|
| **Intubation** RSI | Airway/Ventilation | 確保通氣 | **PPV 減少回心血流 → Preload↓ → 插管後可能 Arrest** |
| **CPAP/BiPAP** | Oxygenation + Preload | 撐開肺泡 + 減回心血量 | **肺水腫首選**，比插管更快更安全 |

</div>

## 關鍵決策示警

<div style="display: grid; grid-template-columns: 1fr 1fr; gap: 1rem; font-size: 0.88em;">

<div>

<div class="alert-red" style="margin-bottom: 0.5rem;">

🚑 **插管前低血壓**

PPV 會進一步降低血壓。
務必先 Fluid 或 Push-dose epi / Norepi
**把 BP 拉住再插管。**

</div>

<div class="alert-red">

🚑 **酸中毒病人插管**

病人靠 hyperventilation 維持 pH。
插管後通氣不足 → **pH 驟降 → Arrest**

設定 RR ≥ 插管前自主呼吸頻率。

</div>

</div>

<div>

<div class="alert-amber">

🚑 **RSI in Shock：劑量調整**

**鎮靜劑減半**（如 Ketamine 2→1 mg/kg）
- CO↓ → 藥物集中中央循環 → 效果更強
- 低血壓副作用在休克病人更致命

**肌鬆劑維持或略增**標準劑量
- 確保一次成功插管，避免反覆嘗試

</div>

Ref: Walls & Murphy, <em>Manual of Emergency Airway Management</em>, 6th Ed; NAEMSP/AMLS Guidelines
</div>

## ROSC 後前 5 分鐘

<div style="display: grid; grid-template-columns: 1fr 1fr; gap: 1rem; font-size: 0.88em;">

<div>

<div class="alert-blue">

**氧合目標**

SpO₂ **90–98%**<br>
從 100% FiO₂ 開始，穩定後滴定下調<br>
通氣：避免 hyperventilation，PaCO₂ 35–45 / EtCO₂ 接近正常<br>
⚠️ 避免過度氧化（Hyperoxia）

</div>

<div class="alert-green">

**血壓目標**

MAP **≥ 65 mmHg**<br>
血壓低 → 繼續 / 啟動 Norepi drip<br>
⚠️ 避免低血壓（Hypotension）

</div>

</div>

<div>

<div class="alert-amber">

**Epi 蓄積警戒**

ROSC 前每 3–5 min 給 Epi<br>
ROSC 後可能積累 3–5 mg<br>
→ 突發高血壓 / 心搏過速 / 心律不整<br>
→ 密切監測，勿再追加

</div>

<div class="alert-red">

**移交清單**

① 12-lead ECG — 排除 STEMI<br>
② 血糖 — 目標 140–180 mg/dL<br>
③ 通報接收醫院（可能 PCI）<br>
④ 避免發燒（> 37.7°C 有害）<br>
⑤ 依病因考慮 coronary evaluation / echo / CT

</div>

</div>

## 模組總結

<div style="display: grid; grid-template-columns: 1fr 1fr; gap: 1rem; font-size: 0.9em;">

<div>

**① 區分 Oxy vs Vent**
SpO₂ 低→給氧；EtCO₂ 高/意識差→給通氣
