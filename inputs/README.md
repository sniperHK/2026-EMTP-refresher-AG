# inputs/

本資料夾存放外部提供的原始素材（例如未整理的簡報、圖片、表格、會議整理檔）。

## 規則

- `inputs/` 內容預設不進 Git（避免大型檔、敏感或未授權素材進版控）
- 保留此 `README.md` 供協作者了解用途與命名慣例
- 建議以日期或來源命名子資料夾，例如：
  - `inputs/2026-02-26_city-protocol/`
  - `inputs/from-instructor/`

## 建議做法

- 若素材轉為可公開引用之參考資料，可整理到 `references/`
- 若為產出結果（PDF/PPTX 匯出），請放 `exports/`
