# M09 台北市 ALS / 休克流程對接表

> 此檔暫沿用舊檔名 `M02_protocol-dosing-map.md`，但用途已改為 **M09 在地流程對接表**。
> 用途：把本課程的病生理學、藥理學與情境案例，對回台北市 EMTP 現場真正使用的作業程序。
> 狀態：已對齊 **115年1月修訂 第六版** SOP 及單項技術規範。正式授課前，仍應以臺北市消防局 / 醫療指導醫師核准的最新版為準。

## 台北市流程重點速查

### 1. ALS / 危急個案（G5）— 115年 v6

台北市手冊把下列**四項指標**列為危急個案（ALS case）定義，適合直接當成本課程「什麼時候要切換成 ALS 腦袋」的起手式：

#### 成人危急個案（G5-1）

**一、生命徵象指標：**

- (1) 意識不清（GCS < 13）
- (2) 呼吸次數 < 10 或 ≧ 30，或未給氧時血氧值 < 90%
- (3) 脈搏 ≧ 150/min 或 < 50/min
- (4) 收縮壓 ≧ 220 或 < 90 mmHg
- (5) CRT ≧ 2 秒
- (6) 體溫 ≧ 40°C 或 < 32°C
- (7) 血糖 < 60 mg/dL（或顯示 low）或 ≧ 500 mg/dL（或顯示 high）
- (8) 發紺

**二、傷病部位指標：** 燒燙傷（2度≧18% 或 3度≧10%）、大量皮下氣腫、截肢、穿刺傷、腦組織/內臟外露、骨盆骨折等

**三、傷病機轉指標：** 高處墜落（≧6m）、高能量撞擊（車速≧50km/h）、爆炸傷等

**四、特殊病患及狀況：**

- (1) 疑似急性腦中風（符合 M6 程序）
- (2) 缺血性胸痛發作（符合 M7 程序）
- (3) 突然或近期昏迷
- (4) 抽搐不止
- (5) 中毒可能危及生命
- (6) 急產
- (7) 情況異常或不穩定之嬰兒或兒童
- (8) 毒蛇咬傷
- **(9) 創傷病患持續出血且有出血性疾病或服用抗凝血劑**（v6 新增）

#### 小兒危急個案（G5-2）— 與成人不同之處

| 指標 | 成人 | 小兒 |
|---|---|---|
| SpO2 | < 90% | **< 94%** |
| 脈搏 | ≧150 或 <50 | <3月: <110 或 ≧170；3月–3歲: <90 或 ≧150；**≧3歲: <60 或 ≧130** |
| 血壓 | SBP < 90 或 ≧220 | **SBP < [70+(2×年齡)]**（1–10歲） |
| 體溫 | ≧40°C 或 <32°C | ≧40°C 或 **<36°C** |

教學上，這一段比單純背「哪支藥多少 mg」更前面，因為它決定：

- 這是不是 ALS 視角的病人
- 是否要提早叫援、提早通報、提早準備進階處置
- 注意 v6 新增：創傷出血合併出血性疾病或抗凝血劑使用者也列為危急個案

### 2. 呼吸困難流程（M2）— 115年 v6

v6 流程以「氧合與呼吸工作量管理」為主軸，藥物只在特定分支出現：

**流程關鍵判斷點：**

1. 初步評估與急救 + 病史詢問
2. **SpO2 < 90% 且呼吸有哮鳴音？** → 進入呼吸窘迫分支
3. 否 → 建立輔助呼吸道及抽吸，氧氣治療維持 **SpO2 ≧ 94%**

**氧氣設備與流量（單項技術 #22）：**

| 設備 | 流量 |
|---|---|
| 鼻導管 | 1–6 L/min |
| 簡單型面罩 | 6–10 L/min |
| 氣切面罩 | 6–10 L/min |
| 非再吸入型面罩 | 10–15 L/min |

**藥物分支（流程中出現的位置）：**

- 步驟 6：協助病人使用自備的吸入型短效**支氣管擴張劑**（MDI / DPI）；<mark>P</mark> 給予氣霧治療
- 步驟 9（嚴重呼吸困難）：持續高濃度氧氣 + 必要時醫療指導下給予 ***腎上腺素 0.3 mg 皮下注射*** + 氣霧治療

**注意：** 步驟 9 的 Epi 0.3 mg SC 是在「病情嚴重」路徑才出現，不是呼吸困難的常規處置。

這塊最適合對接：

- `M01` 的 Oxy vs Vent
- `S01` 張力性氣胸與 `S05` 急性肺水腫的呼吸表現
- 講義裡的 oxygenation / ventilation 區分

### 3. 過敏反應 / 過敏性休克（M5）— 115年 v6

v6 手冊明確把這塊分成「穩定」與「不穩定」：

- **穩定：** 只有輕度呼吸困難，收縮壓仍維持 ≧ 90 mmHg，皮膚表現未呈現休克樣
- **不穩定：** 意識不清、皮膚濕冷、嚴重的呼吸窘迫、疑似上呼吸道阻塞（如呼吸道喘鳴音）或休克徵候（如收縮壓 < 90 mmHg）

**流程重點：**

1. 必要時建立輔助呼吸道及抽吸，氧氣治療維持 SpO2 ≧ 94%（高濃度純氧面罩 15 L/min）
2. <mark>P</mark> 接上 EKG monitor
3. 考慮建立靜脈路徑
4. 穩定 → 依初步評估處置；**不穩定 →** 進入步驟 7

**不穩定路徑（步驟 7）：**

- 快速輸注生理食鹽水（可考慮 2 條大口徑靜脈管路並予以加壓輸入）
- BVM 人工呼吸
- <mark>P</mark> **Epinephrine**（1mg/1ml/1amp，1:1000 製劑）：
  - 成人與青少年：**0.3 mg IM**
  - 小兒：**0.01 mg/kg IM**（至多 0.3 mg）
  - 若不穩定狀況持續，可以**每 10 分鐘重覆一次**上述劑量
- 必要時依【A1 成人氣管內管插管】予以插管（需 Capnography 確認）

**v6 特殊規定：** 若現場為山區郊外，有配置注射型腎上腺素之分隊，**EMT2 可於現場施打一次**，並於到院後與醫護人員交接。

對課程的意義：

- `S03` 不只是「Epi 背劑量」，而是要學會辨識哪一種病人已經進入休克 / 呼吸衰竭
- `M02` 應把 Epi IM 放在流程脈絡裡教，而不是孤立成濃度題
- 注意小兒劑量（0.01 mg/kg）與重複頻率（每 10 分鐘）是考試常見考點

### 4. 急性肺水腫、低血壓及休克（M10）— 115年 v6

這是 M09 最核心的段落。v6 手冊呈現的核心不是藥單，而是**判斷邏輯**：

**休克的定義（v6 註1）—— 組織灌流不足的多種表現：**

1. 血壓降低（SBP < 90 mmHg）
2. 心跳變快（HR ≧ 100，或心跳數值 ≧ SBP）或脈搏變微弱
3. 呼吸變快或變深（可能有代謝性酸中毒）
4. 意識混亂或昏沉
5. 膚色蒼白、皮膚溼冷或出現紫斑
6. CRT ≧ 2 秒
7. 尿量變少

**急性肺水腫徵象（v6 註2）：** 明顯**呼吸急促**（可能有胸痛），合併**咳嗽及帶有血絲的泡沫痰液**、端坐呼吸、發紺、頸靜脈怒張、聽診有肺囉音

**體液（Volume）過多徵象（v6 註4）：** 端坐呼吸、頸靜脈怒張、聽診有肺囉音、下肢水腫

**M10 完整判斷流程（Rate / Volume / Pump）：**

```
步驟 3: SBP < 90 或有「休克」徵象？
  ├─ EMT2 方向 ──→ 步驟 4: 急性肺水腫？
  │                    ├─ 是 → 步驟 5（見下方 NTG 分支）
  │                    └─ 否 → 步驟 6 (Rate): 頻脈或緩脈？
  │                              ├─ 是 → 步驟 7: 參考【M8 緩脈與頻脈】
  │                              └─ 否 → 步驟 8 (Volume): 血量/體液問題？
  │                                        ├─ 是 → 步驟 9: 靜脈注射
  │                                        └─ 否 → 步驟 10 (Pump): 心臟問題？
  │                                                  ├─ 是 → 步驟 11: 輸液試驗（見下方）
  │                                                  └─ 否 → 步驟 12: *Epi IV infusion
```

**急性肺水腫分支（步驟 5）：**

- 嚴密監視並維持 SpO2 ≧ 94%
- 半坐臥方式送醫
- <mark>P</mark> 若無禁忌症，**NTG 舌下錠**：
  - 條件：收縮壓 **≧ 90 mmHg**
  - 頻率：每 **5 分鐘**給 1 次，**最多 3 次**
  - 禁忌注意：脈搏 < 50 或 ≧ 100 次/分時**應小心使用**、**48 小時內使用過壯陽藥物不可服用**
- 必要時依【A1 成人氣管內管插管】予以插管

**低血壓 / 休克分支：**

- 步驟 9：靜脈注射（無體液過多徵象時）
- 步驟 11：輸液後若血壓無明顯反應且**無肺水腫徵象**（呼吸淺快、端坐呼吸、頸靜脈怒張、聽診有肺囉音），再給予靜脈輸液
- 步驟 12：<mark>P</mark> ***Epinephrine IV infusion 0.1–0.5 mcg/kg/min***

**Dirty Epinephrine Drip 泡製（v6 註5/註6）：**

- 適應條件：收縮壓 **< 70 mmHg** 且後送時間較長，經**線上醫療指導同意**
- 泡製：1 支 Epinephrine 加入 **N/S 500 ml**（= 2 mcg/ml）
- 起始速度：調整點滴滴數至**每秒 1 滴**（此時劑量為 **6 mcg/min**；即 0.1 mcg/kg/min @60 kg）
- 到院前應完成輸液瓶罐外標示，到院後主動交接使用劑量及滴注時間

可直接放進課程的在地重點：

| 情境 | 台北市 v6 流程重點 | 課程對接 |
|---|---|---|
| 急性肺水腫 / SCAPE 樣貌 | 半坐臥、SpO2≧94%、辨識肺水腫徵象、NTG（SBP≧90，每5分鐘，最多3次） | 對接 `S05`、`Nohria-Stevenson`、M01 心因性休克 |
| 低血壓但無 overload 徵象 | IV 輸液 → 無反應再追加（步驟 9/11） | 對接 `Pump-Pipe-Tank` 的 `Tank` 與 `Volume` 思維 |
| 極低血壓且後送久 | 線上醫療指導 + Dirty Epi Drip（1支Epi/NS500ml，每秒1滴起） | 對接 M02 升壓藥，但定位為流程分支，不是主軸 |
| Pump failure | <mark>P</mark> Epi IV infusion 0.1–0.5 mcg/kg/min | 對接 M01 心因性休克 |
| 休克分類 | 不是只看血壓：HR、RR、意識、膚色、CRT、尿量 | 對接 M01 休克病生理 |

### 5. OHCA 與 5H5T（M1 成人非外傷無脈搏）— 115年 v6

v6 手冊在無脈搏急救段落強調：

- **高品質 CPR 優先**：用力壓、快快壓、胸回彈、莫中斷、勿過度換氣
- 同步考慮 **5H5T**：Hypovolemia、Hypoxia、Hydrogen ion (acidosis)、Hypo-/Hyperkalemia、Hypothermia、Tension pneumothorax、Thrombosis (coronary)、Thrombosis (pulmonary)、Tamponade (cardiac)、Toxins
- 雙軌救護時，EMT2/EMTP 應依流程進度充分溝通後續決策

**v6 藥物重點（<mark>P</mark> EMTP 項目）：**

| 藥物 | 時機 | 劑量 |
|---|---|---|
| Epinephrine | 每 3–5 分鐘重複給予 | 標準劑量（VF/VT 與 Asystole/PEA 路徑皆有） |
| Amiodarone | 可電擊心律（VF/pVT），於上列第 1 劑藥物給予後 | 第一劑 **300 mg**；約 5 分鐘後可給第二劑 **150 mg** |

**ROSC 判斷標準（v6 註9）：**

1. 有脈搏
2. 無胸外按壓時仍有持續性的可自主活動
3. <mark>P</mark> ETCO₂ 漸增且能維持（**> 40 mmHg** 以上）

**離場原則（v6 註8/11）：**

- 可電擊心律：儘速送醫為原則（SGA/BVM、必要時請求水箱車支援搬運）
- 不可電擊心律：維持原案由現場急救團隊評估及完成適當急救處置，並由 **LEADER 決定**適當離開現場送醫之時機

這一段對應：

- `S04 OHCA`
- `pseudoPEA` 的 reversible causes 討論
- `Pump-Pipe-Tank` 與 `5H5T` 的橋接

## 建議的 M09 教學排法

### A. 先講「什麼病人進入 ALS / 危急流程」

- 直接用 `G5` 危急個案門檻
- 把學員從「知道這病很重」拉到「知道什麼叫在地 ALS 啟動點」

### B. 再講「呼吸困難與休克不是分開兩件事」

- 呼吸困難流程：氧合 / 姿勢 / 氧氣設備升級
- 休克流程：Rate / Volume / Pump
- 把 `M01` 的病生理框架對回台北市程序

### C. 再講 2 個高價值分支

- 過敏性休克：穩定 vs 不穩定、Epi IM 在哪裡出現
- 急性肺水腫 / 低血壓 / 休克：NTG、輸液、dirty epi drip 在哪裡出現

### D. 最後才附藥物欄位

藥物在 M09 只保留「流程中真的需要用到的最低限度資訊」：

| 藥物 | 流程位置 | 劑量速查 |
|---|---|---|
| Epi IM | M5 過敏性休克（不穩定） | 成人 0.3 mg；小兒 0.01 mg/kg（至多 0.3 mg）；每 10 min 可重複 |
| Epi SC | M2 呼吸困難（嚴重） | 0.3 mg（需醫療指導） |
| Epi IV infusion | M10 Pump failure | 0.1–0.5 mcg/kg/min |
| Dirty Epi Drip | M10 極低血壓（SBP<70） | 1支/NS500ml，每秒1滴起（6 mcg/min）需線上醫療指導 |
| NTG 舌下 | M10 急性肺水腫（SBP≧90） | 每 5 min 1 次，最多 3 次 |
| Amiodarone | M1 VF/pVT | 第一劑 300 mg；第二劑 150 mg（間隔 5 min） |
| Epi IV push | M1 OHCA | 每 3–5 min 重複 |

其餘藥理機轉、濃度陷阱、泡製細節，仍回 `M02_ALS-pharmacokinetics.md`。
