# M03 補充資料：高風險低頻盲點

> 定位：`10-15 分鐘` 微型案例。不是把新主模組硬塞回課程，而是補複訓最容易漏掉、但一漏就很傷的保護網。

## 1. 模組目標

完成本模組後，學員應能：

1. 在 `現場評估 → 第一印象 → 重點檢查` 階段，主動把高風險盲點放回鑑別診斷。
2. 說出 4 類常見「看起來不像、但其實很危險」的院前鑑別方向：
   - 敗血症（Sepsis）
   - 內分泌 / 代謝異常（Endocrine / Metabolic）
   - 腦中風 / 神經急症（Stroke / Neuro）
   - 中毒（Tox）
3. 在診斷仍不確定時，先做不會錯過致命生理失敗的處置，而不是過早貼標籤。

## 2. 微型案例 1：敗血症不一定一開始就很像感染

### 案例情境

65 歲男性，發燒 2 天，今天變得嗜睡。家屬說「只是吃不下、越來越喘」。到場時皮膚溫熱、HR 128、BP 88/54、RR 30、SpO₂ 92%，沒有明顯胸痛。

### 評估路徑

- **現場評估**：感染病史 + 意識改變 + 溫暖型休克外觀
- **第一印象**：不是單純脫水，像是分佈性休克 / 敗血症
- **重點檢查**：體溫、微血管回充時間、肺部音、感染來源、血糖、EtCO₂
- **主要鑑別診斷**：
  - 敗血症 / 敗血性休克（Sepsis / Septic Shock）
  - 肺炎合併呼吸衰竭
  - 代謝性腦病變
- **鑑別關鍵**：四肢溫暖、呼吸急促、低血壓與意識改變同時存在

### 敗血症診斷標準（Sepsis-3, 2016）

| 定義 | 標準 |
|---|---|
| **Sepsis** | 疑似或確認感染 + **SOFA score ≥ 2**（需抽血，院內用） |
| **Septic Shock** | Sepsis + 輸液後仍需升壓藥維持 **MAP ≥ 65 mmHg** + **Lactate > 2 mmol/L** |

#### 院前篩檢工具：qSOFA（quick SOFA）

無需抽血，3 項中符合 **≥ 2 項**即高度懷疑：

| 項目 | 閾值 |
|---|---|
| 呼吸速率 | **RR ≥ 22** /min |
| 意識改變 | **GCS < 15**（任何程度的意識變化） |
| 收縮壓 | **SBP ≤ 100** mmHg |

> **注意**：Surviving Sepsis Campaign 2021 建議 qSOFA 不應作為唯一篩檢工具（敏感度不夠高），但在院前資源有限的環境下，qSOFA 仍是最實用的快速評估。結合 **感染線索 + qSOFA ≥ 2** 可有效啟動 sepsis alert。

### 院前可執行處置

| 順序 | 處置 | 說明 |
|---|---|---|
| 1 | **氧合支持** | 維持 SpO₂ ≥ 94%，依需要給氧 |
| 2 | **建立 IV/IO** | 儘早建立血管通路 |
| 3 | **輸液復甦** | NS 或 LR **30 mL/kg**（首小時目標），分次 250-500 mL bolus 並評估反應 |
| 4 | **血糖檢測** | 低血糖常見於敗血症，發現即處理 |
| 5 | **EtCO₂ 監測** | 代謝性酸中毒（乳酸）→ EtCO₂↓ + RR↑；可作為灌流改善指標 |
| 6 | **體溫管理** | 記錄體溫（發燒或低體溫均是警訊） |
| 7 | **提早通報** | 啟動 **Sepsis Alert**，通報接收醫院準備抗生素與乳酸檢測 |

> **時間就是器官**：每延遲 1 小時給予抗生素，敗血性休克死亡率增加約 7.6%（Kumar et al., *Crit Care Med* 2006）。院前能做的最重要的事 = **早期辨識 + 輸液 + 提早通報**，讓醫院端在病人到院時即可給予抗生素。

### 教學重點

院前常見錯誤不是「沒診斷出確切感染源」，而是**沒把敗血症放進前段鑑別診斷**。

**記憶口訣**：老人 + 喘 + 意識差 + 溫暖型低血壓 → 不是「只是虛弱」→ 想到 Sepsis → qSOFA → 輸液 + 通報。

## 3. 微型案例 2：內分泌 / 代謝異常會偽裝成中風、休克或中毒

### 案例情境

58 歲女性，糖尿病史，今天早上開始噁心、虛弱、說話慢。到場時 BP 94/60、HR 118、RR 26，皮膚乾，家屬說她昨天一直喊口渴。

### 評估路徑

- **現場評估**：慢性病史 + 脫水外觀 + 意識改變
- **第一印象**：先想代謝問題，不要急著貼「只是虛弱」
- **重點檢查**：血糖、脫水徵象、呼吸型態、藥物史、近期感染或停藥
- **主要鑑別診斷**：
  - DKA / HHS
  - 敗血症合併脫水
  - 類中風表現（Stroke Mimic）
- **鑑別關鍵**：異常呼吸型態、極度口渴、脫水外觀、病程非突然局部神經缺損

### 前兩步處置

1. 血糖檢測要前移，不要放到最後。
2. 把意識改變視為代謝 / 灌流問題，先救生理，再細分病名。

### 教學重點

複訓課常見盲點不是不知道 DKA，而是**沒有把血糖 / 代謝檢查放進第一輪資料收集**。

## 4. 微型案例 3：腦中風 / 神經急症不等於「有偏癱才算」

### 案例情境

72 歲女性，剛剛突然劇烈頭痛、嘔吐，坐在沙發上說「快昏倒」。到場時 BP 214/110、HR 54、RR 20，肢體無明顯偏癱，但反應變慢。

### 評估路徑

- **現場評估**：突發劇烈頭痛 + 嘔吐 + 高血壓
- **第一印象**：高度懷疑神經急症，而不是單純高血壓或暈眩
- **重點檢查**：FAST-ED / 瞳孔 / GCS / 頸部僵硬 / 發病時間
- **主要鑑別診斷**：
  - 腦出血 / 蜘蛛膜下出血（ICH / SAH）
  - 後循環中風（Posterior Circulation Stroke）
  - 繼發於中樞神經事件的高血壓急症
- **鑑別關鍵**：病程突然、爆炸性頭痛、神經學症狀可能很細微

### 前兩步處置

1. 不要因為沒有明顯偏癱就把神經病因排除。
2. 將「突然頭痛 + 嘔吐 + 意識改變」視為時間敏感轉運問題。

### 教學重點

院前最容易漏掉的是**後循環中風 / 蜘蛛膜下出血（SAH）**，因為它們不像教科書裡最典型的 FAST 表現。

## 5. 微型案例 4：中毒常常長得像敗血症、中風或精神科急症

### 案例情境

34 歲男性，家人說突然躁動、大量出汗、瞳孔很大。到場時 HR 146、BP 176/102、RR 32、T 39.2°C，講話混亂，桌上散落不明藥袋。

### 評估路徑

- **現場評估**：環境線索先看藥袋、容器、粉末、酒精、針具
- **第一印象**：交感亢進 / 中毒症候群
- **重點檢查**：體溫、瞳孔、皮膚乾濕、肌張力 / 顫抖、藥物暴露史、心電圖
- **主要鑑別診斷**：
  - 擬交感神經中毒（Sympathomimetic Toxidrome）
  - 血清素症候群（Serotonin Syndrome）
  - 熱傷害（Heat Illness）
- **鑑別關鍵**：高熱 + 躁動 + 大量出汗 + 瞳孔放大，環境線索很重要

### 前兩步處置

1. 先保護人員與病人安全，再處理高熱 / 躁動 / 灌流。
2. 把中毒視為症候群辨識，不要只問「吃了什麼藥」。

### 教學重點

真正致命的不是沒猜中單一藥名，而是**沒先辨認出中毒症候群與高熱 / 交感風暴**。

## 6. 統整表：盲點在補哪個洞？

| 類別 | 常見錯過原因 | 第一個不該漏的資料 | 第一個不該漏的處置 |
| --- | --- | --- | --- |
| 敗血症 | 只看到發燒或只看到喘 | 意識改變 + 低灌流 + 感染線索 | 先救氧合 / 灌流，提早通報敗血症 |
| 內分泌 / 代謝 | 把意識改變當成模糊主訴 | 血糖、脫水外觀、呼吸型態 | 先確認血糖 / 灌流 |
| 腦中風 / 神經 | 沒有偏癱就低估神經病變 | 發病時間、頭痛、嘔吐、GCS、瞳孔 | 當成時間敏感神經急症轉運 |
| 中毒 | 只盯著精神症狀或單一藥名 | 環境線索、體溫、瞳孔、皮膚、ECG | 先處理安全、高熱、灌流與呼吸 |

## 7. 核心提醒

> 這 4 類盲點不是要你記住完整的診治流程，而是提醒自己：**當你在處理休克或心搏停止時，不要把鑑別診斷縮得只剩那幾個熟悉答案。** 多想一個鑑別方向，可能就多救一條命。
