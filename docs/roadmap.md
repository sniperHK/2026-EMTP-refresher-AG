# Roadmap

## 1. 課程定位

- 目標學員：台北市 EMTP 複訓
- 核心主題：ALS 病生理學 + PK/PD（以瀕死案例做決策訓練）
- 建議時數：4 小時（可拆為 2 x 2 小時）
- 主要評量形式：pre-test + scenario discussion + post-test（混合）

## 2. 模組規劃（建議）

1. 病生理學（ALS 常見決策點）
   - 氧合/通氣/換氣（Oxygenation/Ventilation）
   - 循環與灌流（Perfusion）、休克分類與補償失代償
   - 酸鹼與乳酸、CO2 相關生理
2. 藥物動力學/藥效學（PK/PD）
   - Route：IV/IO/IM/IN 的 onset、bioavailability、分布差異
   - 時間軸：onset/peak/duration、重複給藥與累積
   - 特殊族群：老年、肥胖、肝腎功能不全、低灌流/低體溫、心搏停止
3. 以常用藥串場（依在地 protocol 調整）
   - 甦醒/心律不整/休克/氣道鎮靜鎮痛等常用藥：已建立藥理陷阱框架，待填在地劑量

## 3. 交付物清單

- 講師用：每模組講稿大綱 + 關鍵圖表（機轉/時間軸/陷阱）
- 學員用：重點講義（short notes）
- 情境：4 例（每例含 debrief，已完成）
- 題庫：30 題（含解析，已完成 draft）
- 課堂流程腳本：1 份（已完成）

## 4. 目前缺口（優先級）

1. 在地 protocol 劑量/稀釋法尚未填入（僅通用教學版）
2. 題庫尚未按「難度」與「章節目標」標註
3. scenario 尚缺講師評分表（critical actions checklist）
