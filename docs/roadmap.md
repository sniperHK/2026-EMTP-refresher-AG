# Roadmap

## 1. 課程定位

- 目標學員：台北市 EMTP 複訓
- 核心主題：ALS 病生理學 + PK/PD（以瀕死案例做決策訓練）
- 建議時數：4 小時（可拆為 2 x 2 小時）
- 主要評量形式：pre-test + scenario discussion + mini OSCE + post-test（混合）

## 2. 模組規劃（建議）

1. 病生理學（ALS 常見決策點）
   - 氧合/通氣/換氣（Oxygenation/Ventilation）
   - 循環與灌流（Perfusion）、休克分類與補償失代償
   - 酸鹼與乳酸、CO2 相關生理
2. 藥物動力學/藥效學（PK/PD）
   - Route：IV/IO/IM/IN 的 onset、bioavailability、分布差異
   - 時間軸：onset/peak/duration、重複給藥與累積
   - 特殊族群：老年、肥胖、肝腎功能不全、低灌流/低體溫、心搏停止
3. 以常用藥串場（依在地 protocol 調整）
   - 甦醒/心律不整/休克/氣道鎮靜鎮痛等常用藥
