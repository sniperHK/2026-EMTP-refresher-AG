# S03 — 過敏性休克：當血管背叛了你

> 對應模組：M01 病生理學 ｜ 教學形式：課堂討論
> **與 S02 對比閱讀**：同為低血壓，但機轉完全不同

## 學習重點

- 理解過敏性休克的「雙重打擊」：血管擴張（分布性休克）＋ 呼吸道阻塞
- 與低血容性休克對比：同為低血壓，但 skin sign、JVD、治療方向完全不同
- 理解 Epinephrine 在過敏性休克中的多重藥理作用（α + β1 + β2）

## 情境摘要

午後，119 接獲通報：一名 35 歲女性在公園被蜂螫後全身起疹子、呼吸困難。到場時病人坐在長椅上，全身蕁麻疹，抱怨喉嚨緊緊的、吞嚥困難。初始生命徵象看起來是「低血壓 + 心跳快」——跟出血性休克一樣？但仔細看，這個病人的皮膚是**溫暖潮紅的**。

## 初始資訊

| 項目 | 內容 |
|---|---|
| **Dispatch** | 蜂螫後過敏反應／呼吸困難 |
| **年齡/性別** | 35 歲 女性 |
| **主訴** | 「喉嚨好緊、吞不下去、全身癢、頭很暈」 |
| **病史** | 無已知過敏史（首次嚴重蜂螫反應） |
| **現場觀察** | 右手臂可見蜂螫痕跡 2 處 |

### 初始生命徵象（T0：到場）

| 指標 | 數值 |
|---|---|
| 意識 | E4V5M6（清醒但焦躁，聲音沙啞） |
| 呼吸 | 28 次/min，可聽到**吸氣性喘鳴（stridor）** |
| SpO₂ | 93%（room air） |
| 脈搏 | 118 次/min，規則，脈搏飽滿（bounding） |
| 血壓 | 88/52 mmHg |
| 膚色 | **全身潮紅**、廣泛蕁麻疹、**四肢溫暖** |
| 其他 | 口唇腫脹、舌頭腫大 |

> **教學提示**：低血壓 + 心跳快 → 跟出血性休克像？但皮膚是溫暖潮紅的，不是冰冷蒼白的。為什麼？

## 進程設計（時間軸）

### T+3 min — 給予 Epinephrine IM 前

| 指標 | 數值 |
|---|---|
| 意識 | E4V4M6（說話越來越困難，聲音幾乎聽不到） |
| 呼吸 | 32 次/min，stridor 加重，開始出現**呼氣性哮鳴（wheeze）** |
| SpO₂ | 87% |
| 脈搏 | 130 次/min |
| 血壓 | 76/42 mmHg |
| 其他 | 舌頭嚴重腫脹、流口水 |

> **教學提示**：呼吸道正在關閉（上呼吸道水腫 + 下呼吸道支氣管痙攣）。同時低血壓持續惡化。

### T+5 min — 給予 Epinephrine IM 0.3 mg 後 2 分鐘

| 指標 | 數值 |
|---|---|
| 意識 | E4V5M6（稍改善，較能說話） |
| 呼吸 | 26 次/min，stridor 減輕，wheeze 仍在 |
| SpO₂ | 91%（NRM 15L） |
| 脈搏 | 122 次/min |
| 血壓 | 92/58 mmHg |
| 其他 | 蕁麻疹仍在，但口唇腫脹未再惡化 |

> **教學提示**：Epi 起效了！但還沒完全穩定。Epi IM 的 onset 約 5-10 min，peak 約 15-20 min。

### T+10 min — 持續觀察

| 指標 | 數值 |
|---|---|
| 意識 | E4V5M6（可完整對話） |
| 呼吸 | 22 次/min，stridor 消失，輕微 wheeze |
| SpO₂ | 96%（NRM 15L） |
| 脈搏 | 104 次/min |
| 血壓 | 102/66 mmHg |
| 其他 | 蕁麻疹開始消退 |

> **教學提示**：改善中，但過敏性休克有 biphasic reaction（雙相反應）風險，需持續監測。

## 討論問題

1. **S02（出血性休克）和 S03（過敏性休克）都是「低血壓 + 心跳快」——你怎麼區分？**（提示：皮膚徵象是關鍵線索）

| 比較項目 | 出血性休克（S02） | 過敏性休克（S03） |
|---|---|---|
| 皮膚 | 蒼白、冰冷、濕黏 | **潮紅、溫暖** |
| 脈搏品質 | 微弱 | **飽滿（bounding）** |
| JVD | 無（容積不足） | 無（容積相對不足） |
| 呼吸道 | 通常正常 | **stridor / wheeze** |
| 根本原因 | 血量真的少了 | 血管擴張 → 容器變大了 |

2. **為什麼過敏性休克的皮膚是溫暖的？出血性休克卻是冰冷的？**（提示：血管狀態不同）
3. **Epinephrine 為什麼是過敏性休克的首選藥物？它同時解決了哪些問題？**
4. **如果只給 IV fluid 不給 Epi，能解決過敏性休克嗎？為什麼？**
5. **Epi IM 給了一劑後改善但 5 分鐘後又惡化——你會怎麼做？**（提示：重複給藥時機）

## Debrief 重點

### 病生理機轉：免疫系統的過度反應

```
過敏原（蜂毒）→ IgE 介導 → 肥大細胞大量釋放組織胺等介質
  ↓
三重打擊同時發生：
┌──────────────────┬──────────────────┬──────────────────┐
│  血管效應          │  呼吸道效應        │  皮膚效應          │
│ • 全身血管擴張     │ • 喉頭水腫         │ • 蕁麻疹          │
│ • 微血管通透性↑   │  （上呼吸道阻塞）   │ • 血管性水腫       │
│ • 血漿外滲        │ • 支氣管痙攣       │ • 潮紅            │
│ → 有效循環量↓     │  （下呼吸道阻塞）   │                  │
│ → 分布性休克      │ → 窒息            │                  │
└──────────────────┴──────────────────┴──────────────────┘
  ↓                    ↓
 低血壓 + 低血氧 → 心肌缺氧 → 心搏停止
```

### Epinephrine 的三重藥理：一藥解三毒

| 受體 | 作用 | 對應過敏性休克的問題 |
|---|---|---|
| **α1** | 血管收縮 | 逆轉血管擴張 → 升壓、減少黏膜水腫 |
| **β1** | 心肌收縮力↑、HR↑ | 增加心輸出量 |
| **β2** | 支氣管擴張 | 逆轉支氣管痙攣 → 改善呼吸 |

**這就是為什麼 Epi 是首選——沒有其他單一藥物能同時處理三個問題。**

### 出血性 vs 過敏性：治療邏輯完全不同

| 策略 | 出血性休克 | 過敏性休克 |
|---|---|---|
| **根本治療** | 止血（手術） | Epinephrine |
| **輸液** | 補充「真的少掉的」容量 | 補充「從血管滲出去的」容量 |
| **升壓藥** | 通常不用（血管已最大收縮） | Epi（解決「血管太鬆」問題） |
| **呼吸道** | 通常不是主要問題 | **最致命的威脅之一** |

### Protocol 連結

- **Epinephrine IM**：0.3-0.5 mg（1:1000），大腿外側。可 5-15 min 重複給予
- **IV/IO fluid**：NS/LR 快速 bolus（補充滲出的血漿容量）
- **高流量氧氣**：NRM 15L，若呼吸道阻塞考慮 BVM 或進階呼吸道
- **注意**：biphasic reaction 可在 1-72 小時後再發，到院後需觀察
