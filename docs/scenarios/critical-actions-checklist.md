# Scenarios Critical Actions Checklist（講師評分表）

> 適用情境：`S01_tension-pneumothorax.md`、`S02_hypovolemic-shock.md`、`S03_anaphylactic-shock.md`、`S04_OHCA-VF.md`、`S05_cardiogenic-shock-APE.md`

## 評分方式

- 每項 0-2 分
- `0`：未執行 / 方向錯誤
- `1`：有執行但時機或完整度不足
- `2`：正確且及時執行
- 建議通過門檻：
  - 總分 `>= 80%`
  - 下列 `Critical` 項目不得為 0 分

## 共用項目（五案例皆評）

| 項目 | Critical | 分數 (0/1/2) |
|---|---|---|
| 1. 初始評估與病人安全（scene safety + primary survey） |  |  |
| 2. 能正確辨識主要生理問題（呼吸/循環/代謝） | `Yes` |  |
| 3. 選擇正確監測（SpO2, EtCO2, ECG, BP 趨勢） |  |  |
| 4. 優先處置順序正確（先救命，再精細化） | `Yes` |  |
| 5. 團隊溝通清楚（closed-loop communication） |  |  |
| 6. 能說明處置背後病生理理由（不是只背流程） | `Yes` |  |

## S01 張力性氣胸（Scenario-specific）

| 項目 | Critical | 分數 (0/1/2) |
|---|---|---|
| 辨識單側呼吸音消失 + 低血壓 + JVD 的阻塞性休克圖像 | `Yes` |  |
| 不把問題誤判為單純喘或單純低氧 | `Yes` |  |
| 依流程及時執行減壓策略（依在地 protocol） | `Yes` |  |
| 後續持續監測氧合/灌流變化並調整處置 |  |  |

## S02 低血容性休克（Scenario-specific）

| 項目 | Critical | 分數 (0/1/2) |
|---|---|---|
| 早期辨識「代償性休克」而非等到低血壓才處理 | `Yes` |  |
| 優先止血與循環復甦（而非先插管/鎮靜） | `Yes` |  |
| 輸液策略合理，避免過度升壓導致再出血 | `Yes` |  |
| 能描述需要快速轉送與 definitive hemorrhage control |  |  |

## S03 過敏性休克（Scenario-specific）

| 項目 | Critical | 分數 (0/1/2) |
|---|---|---|
| 辨識分布性休克 + 呼吸道威脅（stridor/wheeze） | `Yes` |  |
| 及時給予 Epinephrine IM（濃度/途徑正確） | `Yes` |  |
| 後續評估是否需重複給藥（5-15 分鐘窗口） | `Yes` |  |
| 併行氧氣、輸液、氣道監測與轉送決策 |  |  |

## S04 OHCA-VF（Scenario-specific）

| 項目 | Critical | 分數 (0/1/2) |
|---|---|---|
| 高品質 CPR（深度/速率/回彈/中斷最少化） | `Yes` |  |
| 電擊時機與循環節奏正確 | `Yes` |  |
| Epinephrine / Amiodarone 給藥時機正確 | `Yes` |  |
| 能同步思考 H's and T's（可逆原因） | `Yes` |  |
| ROSC 後前 5 分鐘目標正確（SpO₂ / 通氣 / MAP / 12-lead） | `Yes` |  |

## S05 心因性休克合併急性肺水腫（Scenario-specific）

| 項目 | Critical | 分數 (0/1/2) |
|---|---|---|
| 辨識 SCAPE（高血壓 + 雙側囉音 + 端坐呼吸）→ Warm+Wet | `Yes` |  |
| 正確處置 Warm+Wet：NTG SL + CPAP + 坐姿（非大量輸液） | `Yes` |  |
| 辨識 Warm+Wet → Cold+Wet 轉折（SBP < 90 + 末梢冰冷） | `Yes` |  |
| Cold+Wet 時立即停 NTG 並轉向升壓策略（push-dose epi） | `Yes` |  |
| 能區分 PseudoPEA vs True PEA（不啟動標準 ACLS 1mg Epi） |  |  |
| 能說明 S05 vs S02 同為低血壓但處置相反的病生理原因 |  |  |
