# S02 — 低血容性休克：代償的極限與失代償的懸崖

> 對應模組：M01 病生理學 ｜ 教學形式：課堂討論

## 學習重點

- 理解出血性休克從代償期到失代償期的生理變化時間軸
- 辨識「血壓還正常」不等於「病人穩定」——代償性心搏過速的意義
- 連結 Class I–IV 出血分級與生命徵象變化的對應關係

## 情境摘要

凌晨 3 點，119 接獲通報：夜市收攤後一名 45 歲男性攤販被刺傷腹部，倒在路邊。到場時病人意識清楚、看起來「還好」，但腹部有一處穿刺傷持續滲血。在轉送過程中，病人從「看起來還好」快速惡化到意識不清、血壓量不到。

## 初始資訊

| 項目 | 內容 |
|---|---|
| **Dispatch** | 刺傷／腹部穿刺傷／有出血 |
| **年齡/性別** | 45 歲 男性 |
| **主訴** | 「肚子被捅了一刀，很痛」 |
| **機轉** | 遭不明人士持刀刺傷左下腹，約 20 分鐘前發生 |
| **現場觀察** | 地面可見約 300 mL 血跡；傷口約 3 cm，持續滲血 |

### 初始生命徵象（T0：到場）

| 指標 | 數值 |
|---|---|
| 意識 | E4V5M6（清醒，可對答，略顯焦慮） |
| 呼吸 | 22 次/min |
| SpO₂ | 98%（room air） |
| 脈搏 | 108 次/min，規則，脈搏略弱 |
| 血壓 | 122/88 mmHg |
| 膚色 | 蒼白、四肢末端微涼、CRT 3 秒 |
| 腹部 | 左下腹穿刺傷，腹部稍硬，壓痛(+) |

> **教學提示**：血壓正常、SpO₂ 正常——但 HR 108 + 蒼白 + CRT 延長 = 身體已經在代償了。

## Assessment Pathway（開場 30-60 秒）

- **Scene size-up**：腹部穿刺傷 + 持續出血，先假設存在內出血
- **First impression**：這不是「穩定外傷」，而是 compensated hemorrhagic shock
- **Focused exam**：皮膚溫度、CRT、脈搏品質、腹部膨隆、意識與 HR / BP 趨勢
- **Leading DDx**：腹腔內出血、骨盆/大血管損傷、其他創傷性低灌流
- **Discriminator**：蒼白冰冷 + CRT 延長 + 腹部膨隆，比當下血壓更能定義這是正在失血的病人

## 進程設計（時間軸）

### T+5 min — 救護車上，建立 IV

| 指標 | 數值 |
|---|---|
| 意識 | E4V5M6（焦躁加重，反覆問「我會不會死」） |
| 呼吸 | 26 次/min |
| SpO₂ | 97% |
| 脈搏 | 124 次/min |
| 血壓 | 110/80 mmHg |
| 其他 | 腹部更加膨脹，壓痛範圍擴大 |

> **教學提示**：HR 從 108 → 124，但 BP 只微降。這是代償在撐住。腹部膨脹 = 腹腔內出血持續中。

### T+10 min — 代償即將崩潰

| 指標 | 數值 |
|---|---|
| 意識 | E3V4M6（嗜睡，問話才回應，回答變慢） |
| 呼吸 | 30 次/min，淺快 |
| SpO₂ | 94% |
| 脈搏 | 138 次/min，微弱、已難摸到橈動脈 |
| 血壓 | 82/60 mmHg |
| 膚色 | 明顯蒼白、全身冷汗、四肢冰冷 |

> **教學提示**：這就是「失代償的懸崖」——血壓突然掉下來，代表已失血 30-40%。

### T+14 min — 失代償

| 指標 | 數值 |
|---|---|
| 意識 | E2V2M4（昏迷，僅對疼痛有反應） |
| 呼吸 | 36 次/min → 逐漸變慢 |
| SpO₂ | 82% |
| 脈搏 | 150 次/min → PEA |
| 血壓 | 56/— mmHg → 無法量測 |

## 討論問題

1. **T0 時 BP 122/88 + HR 108**：如果只看血壓，你會覺得病人穩定嗎？有哪些線索告訴你「不對勁」？
2. **為什麼出血到一定程度後，血壓會「突然」崩掉，而不是慢慢掉？**（提示：代償機轉有極限）
3. **腹部穿刺傷的出血看起來「只有滲血」，但腹部在膨脹——這暗示什麼？**
4. **在院前環境中，針對出血性休克你能做什麼？不能做什麼？**（EMT-P scope 內的處置有哪些？）
5. **給了 1000 mL NS 後血壓短暫回升又掉下來，為什麼？晶體液的限制在哪？**

## Debrief 重點

### 病生理機轉：出血性休克的代償三部曲

```
急性出血 → 循環血量↓
  ↓
第一道防線：交感神經啟動（秒級）
  • HR↑ → 維持 CO
  • 周邊血管收縮 → 維持 MAP
  • 皮膚/腸道灌流犧牲 → 優先供應腦、心
  ↓
第二道防線：體液轉移（分鐘-小時級）
  • 組織間液回流血管 → 部分補充血容量
  • 腎臟留鈉留水
  ↓
代償極限（約失血 30-40%）
  • 交感已全開 → HR 無法再快
  • 血管已最大收縮 → 沒有餘裕
  ↓
失代償：一旦跨過臨界點 → 血壓「斷崖式」下降
  • 組織缺氧 → 乳酸堆積 → 酸中毒
  • 酸中毒 → 心肌抑制 + 血管擴張
  • → 惡性循環 → PEA / 心搏停止
```

### 為什麼「血壓正常」不等於「穩定」？

| 出血分級 | 失血量 | HR | BP | 意識 | 線索 |
|---|---|---|---|---|---|
| Class I | <15% (<750mL) | 正常-微快 | 正常 | 正常 | 幾乎無徵兆 |
| Class II | 15-30% (750-1500mL) | 100-120 | **正常** | 焦躁 | HR↑ + 焦躁 = 紅旗！ |
| Class III | 30-40% (1500-2000mL) | 120-140 | **下降** | 混亂 | 失代償開始 |
| Class IV | >40% (>2000mL) | >140 / 微弱 | 極低 | 昏迷 | 瀕死 |

**本案例的 T0 已在 Class II**——血壓「正常」只是假象。

### 晶體液的限制

- NS/LR 可暫時擴充血管內容量，但**不攜帶氧氣**
- 1000 mL NS 給入後約 20-30 分鐘只有 ~250 mL 留在血管內（其餘滲到組織間隙）
- 出血未止 + 晶體液稀釋 → 稀釋性凝血病變（dilutional coagulopathy）
- **院前重點：止血 > 輸液**

### Protocol 連結

- **止血**：直接加壓、止血帶（四肢）、腹部傷口覆蓋加壓
- **輸液**：IV/IO 建立 → NS/LR（依 protocol 給予適量，避免過度輸液升壓導致再出血）
- **快速轉送**：腹腔內出血在院前無法止血 → **縮短到院時間是最重要的處置**
- **注意**：Class II 以上的出血性休克 = 需要手術止血 + 輸血，院前能做的有限但關鍵
