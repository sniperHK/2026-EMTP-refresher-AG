# S04 — OHCA (VF/VT)：最終共同路徑與復甦的生理學

> 對應模組：M01 病生理學 + M02 PK/PD ｜ 教學形式：課堂討論
> **匯總案例**：前三案的病生理終點都可能走到這裡

## 學習重點

- 理解 VF/VT 的心肌電生理基礎：為什麼心臟「在動卻沒有輸出」
- 連結 CPR、電擊、Epinephrine、Amiodarone 各自針對的病生理環節
- 理解「高品質 CPR」背後的血流動力學：冠狀動脈灌流壓（CPP）的概念
- 回顧前三案：張力性氣胸、出血性休克、過敏性休克若未處置 → OHCA 的共同結局

## 情境摘要

上午 9 點，119 接獲通報：一名 58 歲男性在公園運動時突然倒地，旁人發現時已無反應。目擊者立即呼叫 119 並開始壓胸。EMT-P 到場時旁人已壓胸約 4 分鐘，AED 已貼上但尚未分析。節律顯示 VF。

## 初始資訊

| 項目 | 內容 |
|---|---|
| **Dispatch** | 運動中倒地／無意識無呼吸／旁人 CPR 中 |
| **年齡/性別** | 58 歲 男性 |
| **目擊者陳述** | 「他跑步跑到一半突然抓胸口就倒下去了」 |
| **Downtime** | 約 4 分鐘（目擊倒地，旁人立即打 119 + CPR） |
| **旁人 CPR** | 有壓胸，品質中等（深度約 4 cm、速率不穩） |

### 初始評估（T0：到場）

| 指標 | 數值 |
|---|---|
| 意識 | 無反應 |
| 呼吸 | 無自發呼吸（偶有 agonal gasps） |
| 脈搏 | 頸動脈無脈搏 |
| 節律 | **VF（心室顫動）** |
| AED | 已貼上，提示「建議電擊」 |

## 進程設計（時間軸）

### T0 — EMT-P 到場，確認 VF

**動作**：
- 接手 CPR → 高品質壓胸（5-6 cm、100-120/min、完全回彈）
- 準備電擊

> **討論停頓**：「為什麼 VF 的心臟在心電圖上有波形，但摸不到脈搏？心肌在做什麼？」

### T+1 min — 第一次電擊

| 事件 | 內容 |
|---|---|
| 電擊 | 雙相 200J |
| 電擊後節律 | VF 持續 |
| 動作 | **立即恢復 CPR**（不要停下來看節律）|

> **教學提示**：電擊是讓混亂的電活動「歸零」，但不保證恢復正常節律。為什麼要「電擊後立即壓胸」？

### T+3 min — 2 分鐘 CPR 後再次節律檢查

| 事件 | 內容 |
|---|---|
| 節律 | VF 持續（較細） |
| 電擊 | 第二次雙相 200J |
| 藥物 | **Epinephrine 1 mg IV push** |
| 動作 | 立即恢復 CPR |

> **討論停頓**：「Epi 在心搏停止時做什麼？它不是讓心臟跳起來——它做的是什麼？」

### T+5 min — 第三個循環

| 事件 | 內容 |
|---|---|
| 節律 | VF（仍在）|
| 電擊 | 第三次雙相 200J |
| 藥物 | **Amiodarone 300 mg IV push** |
| 動作 | 恢復 CPR |

> **教學提示**：Amiodarone 不是用來「電擊心臟」的，它解決的是另一個問題——什麼問題？

### T+7 min — 第四個循環

| 事件 | 內容 |
|---|---|
| 節律 | **轉為竇性心搏過速（sinus tachycardia）** |
| 脈搏 | 頸動脈可觸及，微弱 |
| 血壓 | 72/48 mmHg |
| SpO₂ | 88%（BVM 通氣中） |
| 意識 | 無反應，GCS E1V1M3 |

> **教學提示**：ROSC！但這不是結局——病人仍處於 post-cardiac arrest syndrome。

## 討論問題

1. **VF 的心臟其實「在動」——為什麼沒有心輸出量？**（提示：想像心肌細胞各自為政 vs 協調收縮）
2. **高品質 CPR 的「5-6 cm、100-120/min、完全回彈」不是隨意數字——每個參數背後的生理學是什麼？**
3. **Epinephrine 在 CPR 期間的角色不是「讓心臟跳起來」——那它到底在做什麼？**（提示：冠狀動脈灌流壓）
4. **第一次電擊沒成功 → 為什麼要先壓胸 2 分鐘，而不是馬上再電？**
5. **回顧前三個案例（張力性氣胸、出血性休克、過敏性休克）——如果都沒處理，最終都會走到什麼共同結局？各自的「H's and T's」對應是什麼？**

## Debrief 重點

### 病生理機轉：VF 的心肌電生理

```
正常心臟：                    VF 心臟：
SA node → 有序傳導             多個微折返迴路 (micro-reentry)
→ 心房→心室依序收縮            → 心肌細胞各自亂放電
→ 有效泵血                    → 心臟「在蠕動」但無泵血功能
→ 有脈搏                     → 無脈搏、無心輸出量
```

**電擊的目的**：不是「啟動」心臟——是把所有混亂的電活動同時「歸零」（depolarize all cells simultaneously），給 SA node 一個重新主導的機會。

### CPR 的血流動力學：為什麼「品質」那麼重要

```
壓胸 → 胸腔內壓↑ → 血液被擠出心臟 → 暫時性前向血流
  ↓
完全回彈 → 胸腔內壓↓ → 靜脈回流 → 心臟重新充血
  ↓
交替進行 → 產生約正常 25-33% 的心輸出量
```

| 參數 | 生理意義 | 不足的後果 |
|---|---|---|
| 深度 5-6 cm | 足夠壓縮心臟產生前向血流 | 血流不足 → CPP↓ |
| 速率 100-120/min | 最佳化每分鐘心輸出 | 太慢=流量不足；太快=回彈不完全 |
| **完全回彈** | 允許靜脈回流、心臟再充血 | 不回彈=持續壓迫→靜脈回流↓→CPP↓ |
| **中斷最少化** | 維持 CPP（中斷 >10 秒 CPP 歸零） | 每次中斷 CPP 需 60 秒重建 |

### Epinephrine 在 CPR 中的角色：不是「讓心跳起來」

```
Epi 1 mg IV → α1 受體 → 周邊血管收縮
  ↓
主動脈舒張壓↑
  ↓
冠狀動脈灌流壓 (CPP)↑ = 主動脈舒張壓 - 右心房壓
  ↓
缺血的心肌獲得血流 → 心肌能量↑
  ↓
增加電擊成功轉復的機率
```

**注意**：Epi 的 β1 效應（增加心肌氧耗）在心搏停止中反而是**負面的**——但 α1 的升壓效應利大於弊。

### Amiodarone 的角色：穩定電氣環境

- VF 反覆電擊不轉復 = 心肌電氣不穩定（持續的 re-entry circuits）
- Amiodarone：多通道阻斷劑 → 延長不反應期 → 中斷 re-entry → 創造電擊成功的條件
- **不是用來「電擊心臟」的**，是用來「讓電擊更可能成功」的

### 前三案的匯流：H's and T's

| 前案 | 可逆原因 | H's/T's 分類 | 如何逆轉 |
|---|---|---|---|
| S01 張力性氣胸 | 胸腔內壓↑ | **T**ension pneumothorax | Needle decompression |
| S02 出血性休克 | 循環血量↓ | **H**ypovolemia | 止血 + 輸液 |
| S03 過敏性休克 | 血管擴張 + 呼吸道阻塞 | Anaphylaxis（T: Toxins 類別） | Epinephrine |

**如果前三案都沒處理 → 都走到 PEA 或 VF → 就是本案 S04。**
**ACLS 不只是背演算法——是理解「為什麼心臟會停」並在停之前逆轉可逆原因。**

### Protocol 連結

- **CPR**：高品質壓胸、最少中斷
- **電擊**：VF/pVT → 雙相 200J，每 2 分鐘 CPR 循環後重新評估
- **Epinephrine**：1 mg IV/IO，q3-5 min（首次在第二或第三次電擊後給予）
- **Amiodarone**：refractory VF → 300 mg IV push；可追加 150 mg
- **進階呼吸道**：依 protocol 執行氣管插管（EMT-P scope）
- **可逆原因**：時刻思考 H's and T's，尤其前三案的情境
