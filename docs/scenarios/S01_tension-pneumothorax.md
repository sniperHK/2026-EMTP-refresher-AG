# S01 — 張力性氣胸：從呼吸衰竭到阻塞性休克

> 對應模組：M01 病生理學 ｜ 教學形式：課堂討論

## 學習重點

- 辨識張力性氣胸從呼吸衰竭進展到循環衰竭（阻塞性休克）的連續過程
- 理解胸腔內壓升高如何同時影響通氣與靜脈回流
- 連結「氣管偏移／頸靜脈怒張／低血壓」三聯徵背後的病生理機轉

## 情境摘要

某日深夜，119 接獲通報：一名 28 歲男性騎機車自摔，胸部撞擊電線桿。到場時病人意識清楚但明顯喘，抱怨右側胸痛。初始生命徵象尚可，但在轉送途中病情急速惡化——從呼吸困難進展到低血壓、意識改變。

## 初始資訊

| 項目 | 內容 |
|---|---|
| **Dispatch** | 機車事故／單一傷患／胸部鈍傷 |
| **年齡/性別** | 28 歲 男性 |
| **主訴** | 「右邊胸口很痛，吸不到氣」 |
| **機轉** | 騎機車自摔，右胸撞擊電線桿，速度約 40 km/h |

### 初始生命徵象（T0：到場）

| 指標 | 數值 |
|---|---|
| 意識 | E4V5M6（清醒，可對答） |
| 呼吸 | 28 次/min，淺快，右側胸壁活動度↓ |
| SpO₂ | 92%（room air） |
| 脈搏 | 110 次/min，規則 |
| 血壓 | 128/82 mmHg |
| 膚色 | 蒼白，微汗 |

## Assessment Pathway（開場 30-60 秒）

- **Scene size-up**：高能量胸部鈍傷，先假設有單側胸腔機械性問題
- **First impression**：單側外傷後呼吸困難，先朝機械性呼吸障礙與 tension physiology 思考
- **Focused exam**：雙側呼吸音與胸壁起伏、叩診音、氣管位置、JVD、氧氣後趨勢
- **Leading DDx**：張力性氣胸、單純氣胸、血胸/肺挫傷
- **Discriminator**：呼吸音消失 + hyperresonance + JVD / hypotension = 張力性氣胸直到證明不是

## 進程設計（時間軸）

### T+5 min — 救護車上，開始轉送

| 指標 | 數值 |
|---|---|
| 意識 | E4V5M6（焦躁不安，反覆說「吸不到氣」） |
| 呼吸 | 34 次/min，明顯費力，右側**呼吸音消失** |
| SpO₂ | 85%（NRM 15L 下） |
| 脈搏 | 126 次/min |
| 血壓 | 118/78 mmHg |
| 其他 | 右胸叩診**過度共鳴（hyperresonance）** |

> **教學提示**：此時已出現呼吸衰竭徵象，但血壓仍靠代償維持。為什麼？

### T+10 min — 病情惡化

| 指標 | 數值 |
|---|---|
| 意識 | E3V4M5（嗜睡，回答含糊） |
| 呼吸 | 40 次/min → 逐漸變慢、變淺 |
| SpO₂ | 72%（NRM 15L 下） |
| 脈搏 | 140 次/min，微弱 |
| 血壓 | 78/50 mmHg |
| 其他 | **頸靜脈怒張（JVD）**、氣管**向左偏移** |

> **教學提示**：這裡是關鍵轉折——從「呼吸問題」變成「循環問題」。為什麼血壓突然掉了？

### T+13 min — 未處理的結局

| 指標 | 數值 |
|---|---|
| 意識 | E1V1M3（昏迷） |
| 呼吸 | 8 次/min，瀕死呼吸 |
| SpO₂ | 無法偵測 |
| 脈搏 | PEA → 心搏停止 |
| 血壓 | 無法量測 |

## 討論問題

1. **T0 時 SpO₂ 92% + HR 110**：這代表什麼？身體正在做什麼代償？
2. **為什麼 T+5 給了高流量氧氣，SpO₂ 還是持續掉？**（提示：問題不在「缺氧氣」，在於什麼？）
3. **T+10 出現 JVD + 低血壓**：為什麼氣胸會影響到血壓？胸腔內壓升高如何影響心臟？
4. **如果把張力性氣胸當成一般「喘」來處理（只給氧、BVM），結果會如何？**
5. **在院前環境中，你能做什麼來逆轉這個惡性循環？**（依預立醫療流程）

## Debrief 重點

### 病生理機轉：壓力的連鎖反應

```
胸壁損傷 → 肋膜破裂 → 空氣進入胸腔（單向活瓣）
  ↓
胸腔內壓持續上升
  ↓
┌─────────────────────┬─────────────────────┐
│   呼吸效應            │   循環效應            │
│ • 患側肺塌陷          │ • 大靜脈受壓          │
│ • 縱膈腔偏移          │ • 靜脈回流↓           │
│ • 健側肺也被壓迫       │ • 前負荷↓ → CO↓      │
│ → 通氣量↓ → 低血氧    │ → 阻塞性休克          │
└─────────────────────┴─────────────────────┘
  ↓
 兩條路徑匯合 → PEA → 心搏停止
```

### 為什麼「只給氧」不夠？

- 張力性氣胸的低血氧來自**機械性通氣障礙**（肺被壓扁），不是「氧氣不夠」
- 就算 FiO₂ 100%，如果肺無法擴張，氣體交換面積趨近於零
- 必須**解除壓力**（needle decompression 或 chest tube），才能恢復通氣

### 生命徵象的「時間密碼」

| 階段 | 主要問題 | 代償表現 | 何時崩潰 |
|---|---|---|---|
| 早期 | 通氣↓ | HR↑（交感代償）、呼吸加快 | SpO₂ 仍>90% |
| 中期 | 通氣↓ + 靜脈回流↓ | HR↑↑、血管收縮維持 BP | SpO₂ 急速下降 |
| 晚期 | CO↓↓ | 代償失敗 | BP 崩潰、PEA |

### Protocol 連結

- **辨識**：單側呼吸音消失 + 叩診過度共鳴 + JVD + 低血壓 → 高度懷疑張力性氣胸
- **EMT-P 處置**：依預立醫療流程執行 needle decompression（依地方 protocol）
- **關鍵時間**：從辨識到減壓的時間直接決定存活率
- **講師評分**：`docs/scenarios/critical-actions-checklist.md`（S01 區段）
