# S05 — 心因性休克合併急性肺水腫：Pump 壞了，水淹上來

> 對應模組：M01 病生理學（Nohria-Stevenson）+ M02 PK/PD（NTG、Push-dose Epi）｜ 教學形式：15 分鐘精實課堂討論
> **與 S02 對比閱讀**：同為低血壓，但 Tank 空 vs Pump 不力 → 處置完全相反

## 學習重點

- 辨識 SCAPE（Sympathetic Crashing Acute Pulmonary Edema）：高血壓 + 急性肺水腫 + 呼吸窘迫
- 使用 Nohria-Stevenson 矩陣分類：Warm+Wet → Cold+Wet 的轉折與處置改變
- 理解 Forward failure（低灌流）與 Backward failure（肺水腫）同時發生的機轉
- 辨識 PseudoPEA：有節律但無脈搏 → 不是 cardiac arrest，是 extreme shock

## 情境摘要

傍晚，119 接獲通報：一名 65 歲男性在家中突然呼吸困難、胸口壓迫感。到場時病人坐在沙發上喘得很厲害、滿身大汗。初始血壓很高（180/110），雙肺佈滿囉音。

## 初始資訊

| 項目 | 內容 |
|---|---|
| **Dispatch** | 胸痛合併呼吸困難 |
| **年齡/性別** | 65 歲 男性 |
| **主訴** | 「胸口好像被壓住、喘不過氣、冒冷汗」 |
| **病史** | 高血壓、糖尿病、抽菸 30 年 |
| **現場觀察** | 端坐呼吸（不敢躺下）、說話斷斷續續 |

### 初始生命徵象（T0：到場）

| 指標 | 數值 |
|---|---|
| 意識 | E4V4M6（清醒但焦躁、說話費力） |
| 呼吸 | 32 次/min，**雙側明顯囉音（crackles）**，咳出粉紅泡沫痰 |
| SpO₂ | 88%（room air） |
| 脈搏 | 110 次/min，規則 |
| 血壓 | **180/110 mmHg** |
| 膚色 | 面色蒼白、冒冷汗，但四肢尚溫暖 |
| ECG | Sinus tachycardia，ST elevation V1-V4 |

> **教學提示**：高血壓 + 雙側囉音 + 粉紅泡沫痰 + 端坐呼吸 = **SCAPE**。
> Nohria-Stevenson 分類：**Warm+Wet（B 象限）** → NTG + CPAP。
> ECG 提示前壁 STEMI — 這是 AMI 導致的急性肺水腫。

## Assessment Pathway（開場 30-60 秒）

- **Scene size-up**：胸痛 + 呼吸困難 + 粉紅泡沫痰，不要先入為主當成單純氣喘
- **First impression**：先以 Warm+Wet 的 SCAPE / cardiogenic pulmonary edema 思考，但要隨時警覺掉進 Cold+Wet
- **Focused exam**：肺音、末梢溫度、血壓趨勢、12-lead ECG 與 STEMI 線索
- **Leading DDx**：AMI 相關 SCAPE / cardiogenic shock、肺栓塞、非心因性呼吸衰竭
- **Discriminator**：高血壓 + bilateral crackles + pink frothy sputum + STEMI 線索，會把方向拉向 SCAPE

## 進程設計（時間軸）

### T+5 min — NTG SL + CPAP 後突然惡化

**處置已做**：NTG SL 0.4 mg + CPAP 10 cmH₂O + 高流量 O₂

| 指標 | 數值 |
|---|---|
| 意識 | E3V3M5（嗜睡、回答混亂） |
| 呼吸 | 28 次/min，囉音加重，CPAP 下 SpO₂ 僅 85% |
| SpO₂ | **85%** |
| 脈搏 | 130 次/min，微弱 |
| 血壓 | **75/50 mmHg** |
| 膚色 | **四肢冰冷、蒼白、濕黏** |

> **教學提示**：從 Warm+Wet → **Cold+Wet（C 象限）** 的轉折！
>
> - **立即停 NTG**（已經低血壓，再給 NTG = 更低）
> - **不能大量輸液**（肺已經淹水了）
> - **不能給利尿劑**（低灌流的肺水腫不是 volume overload 問題）
> - **需要升壓**：Push-dose Epi 10-20 mcg IV + 考慮 Norepinephrine drip

### T+10 min — 持續惡化至 PseudoPEA

**處置已做**：Push-dose Epi 20 mcg IV，準備 Norepinephrine drip

| 指標 | 數值 |
|---|---|
| 意識 | E1V1M3（幾乎無反應） |
| 呼吸 | 淺而慢，需 BVM 輔助 |
| SpO₂ | 78% |
| 脈搏 | **ECG 顯示 sinus tachycardia 120/min → 但頸動脈脈搏極微弱、幾乎摸不到** |
| 血壓 | 量不到 / 估 < 40 mmHg |

> **教學提示**：**PseudoPEA！**
> - ECG 有 organized rhythm（心臟還在跳）
> - 但脈搏摸不到（心輸出量極低）
> - **這不是 cardiac arrest → 不要啟動標準 ACLS（1 mg Epi 會打爆還在跳的心臟）**
> - **處置**：push-dose epi 10-20 mcg IV q 1-2 min + Norepinephrine drip + 持續 BVM + 快速轉送
> - 若有 POCUS（受訓操作者，不延長 rhythm check 暫停）：心臟有收縮支持 pseudoPEA 診斷 → 持續升壓策略（2025 AHA：intra-arrest POCUS 診斷價值尚未完全建立，作為輔助工具，非必要條件）

**結局**：Push-dose Epi 反覆 2 次（共 40 mcg）+ Norepi drip 啟動 → BP 回到 85/55，脈搏可觸及，SpO₂ 回升至 90%。快速轉送心導管室。

## 快速 Debrief（7 分鐘）

### 三個核心問題

1. **Warm+Wet → Cold+Wet 的轉折點是什麼？你怎麼判斷？**
   - SBP 跌破 90 + 末梢從溫暖變冰冷 = 從 B 象限掉到 C 象限
   - 此時必須停止所有降壓/擴張藥物（NTG），轉向升壓策略

2. **S05 跟 S02 同樣是「低血壓 + 末梢冰冷」，處置為什麼完全相反？**

   | 比較 | S02 低血容性休克 | S05 心因性休克 |
   |---|---|---|
   | Pump-Pipe-Tank | Tank 空 | **Pump 不力** |
   | JVD | 無 | 可能有 |
   | 肺音 | 乾淨 | **雙側囉音** |
   | 處置 | **輸液 + 止血** | **升壓/強心（不能輸液！）** |

3. **T+10 的 PseudoPEA — 為什麼不能啟動標準 ACLS？**
   - 標準 ACLS Epi 1 mg = 休克劑量的 50-100 倍
   - 心臟還在跳 → 全量 Epi 打進去 = 心肌過度興奮 → VF/VT
   - PseudoPEA = extreme shock → 用休克劑量（10-20 mcg）升壓

### Protocol 連結

- **SCAPE 處置**：NTG SL 0.4 mg（可重複）+ CPAP/BiPAP + 坐姿 + 高流量 O₂
- **Cold+Wet 處置**：停 NTG → Push-dose Epi 10-20 mcg IV → Norepinephrine drip → 快速轉送
- **PseudoPEA 處置**：push-dose epi（不是 ACLS 劑量）+ 升壓藥 drip + 找可逆原因（AMI → cath lab）
- **講師評分**：`docs/scenarios/critical-actions-checklist.md`（S05 區段）

> [!NOTE]
> 藥物劑量依據 AHA ACLS 2020 + 2025 ACC Cardiogenic Shock Expert Consensus。台北市在地 protocol 確認後請以在地版為準。
