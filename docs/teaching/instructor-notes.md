# 講師引導筆記（Instructor Facilitation Notes）

> 每個情境提供 Socratic 引導提問、常見學員迷思、教學陷阱提醒。
> 目的：確保 debrief 聚焦在「為什麼」而非「怎麼做」，引導病生理思考。

---

## 統一 Debrief Script 框架（所有情境適用）

> 依據 AHA 2025 Resuscitation Education Science：scripted debriefing 可提升一致性與學習遷移。

每個情境 debrief 必須包含以下四個固定問題，順序不變，每題 1–2 分鐘：

| 順序 | 固定問題 | 目的 |
|:---:|---|---|
| **Q1** | **「現場發生了什麼變化？你注意到什麼？」** | 觀察事實，建立共識 |
| **Q2** | **「是哪個生理機制出了問題？」** | 連結 Pump/Pipe/Tank 或病生理框架 |
| **Q3** | **「你的哪個行動對應到那個失敗？」** | 行動—機轉閉環，強化決策連結 |
| **Q4** | **「如果再來一次，什麼訊號會讓你更早改變策略？」** | 前瞻性思考，轉移到下一個病人 |

**時間管理原則**：
- 每個情境 debrief 以 8 分鐘為上限（S05 壓縮至 7 分鐘）
- 若時間壓力大，優先保留 Q2 + Q3（病生理 + 行動對應）
- 四題問完後視情況開放 1 分鐘自由提問，非強制

**語氣規範**：
- 不說「你做錯了」，說「這個線索出現在哪一個時間點？」
- 不說「應該要這樣」，說「這個選擇背後的生理邏輯是什麼？」

## S01 張力性氣胸（建議 Debrief 8 分鐘）

### 引導提問

1. **「這個病人低血壓的原因跟出血不同在哪裡？」**
   - 引出 Obstructive vs Hypovolemic 的區別
   - 期待答案：胸腔壓力壓迫心臟/大血管 → Tank 被擋住（不是 Tank 空）
   - 追問：「所以我們該輸液還是減壓？」

2. **「減壓之後，你預期生命徵象怎麼變化？多快？」**
   - 引出病生理逆轉的時間感
   - 期待答案：立即改善（秒到分鐘級）— BP↑, SpO₂↑, JVD 消退
   - 追問：「如果減壓後沒改善，你會考慮什麼？」（位置不對、雙側、其他原因）

### 常見迷思

| 迷思 | 修正 |
|------|------|
| 「JVD 一定要看到才能診斷」 | 合併低血容時 JVD 可能不明顯；靠的是「單側呼吸音消失 + 低血壓 + 臨床懷疑」的組合 |
| 「先打 IV 輸液再減壓」 | 阻塞性休克的根本問題不是血容量，輸液只是暫時撐住，**減壓才是 definitive** |

### 教學陷阱

- 別讓討論偏到 needle decompression 的手法細節（穿刺位置、針號）— focus 在「為什麼要做」
- 強調 Pump-Pipe-Tank：Tank 被外力擋住 → 解除外力是唯一正解

## S02 低血容性休克（建議 Debrief 8 分鐘）

### 引導提問

1. **「這個病人 HR 130, BP 還正常——他穩定嗎？」**
   - 引出代償 vs 失代償的分界
   - 期待答案：不穩定！HR 130 代表交感神經在拚命代償，BP 正常是假象
   - 追問：「你怎麼知道他已經在代償？」（HR↑, CRT↑, 焦躁, 皮膚冰冷）

2. **「如果你先插管再止血，會發生什麼？」**
   - 引出「RSI in hypovolemia = death」的概念
   - 期待答案：正壓通氣 → 胸內壓↑ → 靜脈回流↓ → 在已經 Tank 空的病人身上 = cardiac arrest
   - 追問：「所以正確順序是？」（止血 → 輸液 → 穩定循環 → 再考慮呼吸道）

### 常見迷思

| 迷思 | 修正 |
|------|------|
| 「BP 正常就是穩定」 | 代償期 BP 可以正常到 Class II-III 出血（750-1500 mL），等 BP 降就已經失代償 |
| 「先 ABC 所以先插管」 | 在出血性休克，C（止血+輸液）可能比 A（插管）更優先 — 「kill the patient to save the airway」是常見錯誤 |
| 「大量輸液打到 BP 正常」 | Permissive hypotension（SBP 80-90 即可），過度升壓會沖掉凝血塊 → 再出血 |

### 教學陷阱

- 學員容易直接跳到「輸液」，忘記最重要的是「止血」
- 引導反思：「到院前你能控制的出血有哪些？」（直接加壓、止血帶、骨盆固定）

## S05 心因性休克 + APE（建議 Debrief 7 分鐘）

> **注意：S05 只有 15 分鐘，debrief 必須精準聚焦 3 個問題，不要發散。**

### 引導提問

1. **「S05 跟 S02 都是低血壓 + 末梢冰冷，你怎麼區分？」**
   - 引出 Tank 空 vs Pump 不力的鑑別
   - 期待答案：JVD（S05 可能有）、肺音（S05 雙側囉音）、病史（胸痛 vs 外傷出血）
   - 追問：「處置完全相反——具體怎麼不同？」（S02 輸液；S05 升壓/強心，不能輸液）

2. **「Warm+Wet 給 NTG 很合理，為什麼突然要停？」**
   - 引出 Nohria-Stevenson 象限轉折
   - 期待答案：SBP 掉到 75/50 = 已經 Cold+Wet（C 象限），NTG 在 SBP < 90 禁用
   - 追問：「你怎麼知道從 B 象限掉到 C 象限了？」（BP↓ + 末梢從溫暖變冰冷）

3. **「如果脈搏摸不到但 ECG 有心跳，你要壓胸嗎？」**
   - PseudoPEA 核心問題
   - 期待答案：不一定！若有受訓者可在不延長暫停下快速用 POCUS 確認，看到收縮可轉向 pseudoPEA / low-flow shock 思維；若無法快速確認，不要拖慢標準 resuscitation
   - 追問：「為什麼不能打 ACLS 劑量 1 mg？」（心臟還在跳，1 mg 會打成 VF）

### 常見迷思

| 迷思 | 修正 |
|------|------|
| 「肺水腫就是打 Lasix」 | SCAPE 是 afterload 問題（血管太緊），不是 volume overload — Furosemide 不是首選，NTG 才是 |
| 「摸不到脈搏就壓胸」 | 若能在不延長暫停下快速確認 pseudoPEA，壓胸未必是第一優先；若無法快速確認，仍以標準 resuscitation 節奏為主 |
| 「低血壓就要輸液」 | Cold+Wet 輸液 = 把水倒進已經淹水的肺 → 更濕更喘 |

### 教學陷阱

- S05 時間很緊，講師**不要**在 SCAPE 或 Nohria-Stevenson 上展開太多理論，快速帶過然後聚焦 3 個問題
- 務必在 debrief 最後花 30 秒做 S05 vs S02 的對比總結

## S03 過敏性休克（建議 Debrief 8 分鐘）

### 引導提問

1. **「Epi 為什麼要 IM 而不是 IV？如果 IM 沒效呢？」**
   - 引出 IM→IV 的升級邏輯
   - 期待答案：IM 安全（不容易造成心律不整），IV push 風險高但在嚴重/IM 失敗時必要
   - 追問：「休克病人 IM 吸收會有什麼問題？」（周邊血管收縮 → IM 吸收差 → late dump 風險）

2. **「這個病人同時有喘 + 低血壓——你先處理哪個？」**
   - 引出呼吸道 vs 循環的優先序在分布性休克中的思考
   - 期待答案：Epi 同時處理兩個問題（α1 升壓 + β2 支氣管擴張），所以 Epi 是第一線
   - 追問：「如果給了 Epi 呼吸道還是緊，下一步？」（考慮 Albuterol, 準備 airway management）

### 常見迷思

| 迷思 | 修正 |
|------|------|
| 「Epi 打一針就好了」 | 很多嚴重過敏需要 2-3 劑（q5-15min），第一劑沒完全效不代表失敗 |
| 「先打 Diphenhydramine 再看」 | 抗組織胺只處理皮膚症狀，**不處理低血壓和喉頭水腫** — Epi 才是救命藥 |
| 「Epi 1:1000 可以 IV push」 | 1:1000 原液 IV push = 心律不整高風險！IV 需稀釋至 push-dose 10 mcg/mL（1 支 + NS 至 100mL） |

### 教學陷阱

- 別讓「抗組織胺 vs 類固醇」的討論蓋過 Epi 的核心地位
- 強調濃度混淆是致命錯誤（IM 用 1:1000 原液，IV push-dose 需稀釋至 10mcg/mL）

## S04 OHCA-VF（建議 Debrief 10 分鐘）

### 引導提問

1. **「電擊是把心臟電起來嗎？」**
   - 引出歸零 vs 啟動的觀念修正
   - 期待答案：電擊是讓混亂的電活動同時 depolarize → 歸零 → 給 SA node 重新主導的機會
   - 追問：「所以電擊不成功的原因可能是什麼？」（心肌缺血太嚴重，SA node 無法接管）

2. **「Epi 在 CPR 中不是讓心跳起來——那它做什麼？」**
   - 引出 CPP（冠狀動脈灌流壓）+ α1 效應
   - 期待答案：α1 → 周邊血管收縮 → 主動脈舒張壓↑ → CPP↑ → 缺血心肌獲得血流 → 增加電擊成功率
   - 追問：「那 Epi 的 β1 效應呢？」（增加心肌氧耗 → 在 arrest 中其實是負面的，但 α1 利大於弊）

3. **「電擊後變成 organized rhythm 但沒脈搏——下一步？」**
   - PseudoPEA 連結 S05
   - 期待答案：區分 true PEA vs pseudoPEA → POCUS → 有收縮 = pseudoPEA → push-dose epi 10-20 mcg
   - 追問：「如果沒有 POCUS 呢？」（臨床判斷：升壓試探，觀察反應）

### 常見迷思

| 迷思 | 修正 |
|------|------|
| 「電擊失敗就再電」 | 需要 2 min CPR 重建 CPP，心肌有能量才有機會轉復 |
| 「PEA 就是繼續壓」 | 需區分 true vs pseudo — pseudoPEA 的心臟還在跳，管理方式 = 重度休克治療 |
| 「H's and T's 要背出來」 | 重點不是背誦，是連結前四案：S01=Tension, S02=Hypovolemia, S03=Toxins/Anaphylaxis, S05=AMI |

### 教學陷阱

- H's and T's 容易變成背誦比賽 → 引導回「前四案都是可逆原因，如果早期辨識就不會走到 OHCA」
- S04 的 debrief 是整堂課的收尾 → 要把五個情境串起來，讓學員看到「全貌」

## 通用講師提醒

### 每個案例的核心三問（講師必問）
1. 現在壞掉的是哪個生理環節？（Pump / Pipe / Tank）
2. 你選的處置在修正哪個環節？
3. 若處置失敗，下一步怎麼調整？

### Socratic 提問技巧
- **不要直接給答案**：「你覺得呢？」「為什麼你這樣想？」
- **用反例挑戰**：「如果他的肺音是乾淨的，你的判斷會改變嗎？」
- **引導對比**：「S05 跟 S02 哪裡不同？」「Epi IM vs IV 各自什麼時候用？」

### 時間控制
- 每個提問給學員 **30-60 秒思考**，不要馬上接話
- 如果討論發散，用「很好的觀點，我們先記住這個，回到今天的重點——」收回來
- **S05 debrief 只有 7 分鐘**：嚴格控制在 3 個問題，每問 2 分鐘 + 1 分鐘收尾
