## 專案目標

- 產出「台北市 EMTP 複訓」教學素材，聚焦 **ALS（Advanced Life Support）病生理學**與**藥物動力學/藥效學（PK/PD）**

- **PK/PD**：pharmacokinetics / pharmacodynamics
- 路徑用語：`inputs/` 放原始檔；`exports/` 放輸出檔；Markdown 內容放 `docs/`

## 資料夾結構

- `docs/`: 主要教材（Markdown）
- `docs/modules/`: 模組內容（每個主題一檔）
- `docs/assessments/`: 評量藍圖與題庫
- `docs/scenarios/`: 情境案例與 debrief
- `docs/templates/`: 可重用模板
- `docs/teaching/`: 課程執行腳本與講師引導流程
- `references/`: 參考文獻清單（可放 citations、連結、摘要、課程比較分析）
- `references/raw/`: 原始 PDF/圖片/檔案（**不進 Git**；見 `.gitignore`）
- `inputs/`: 外部提供或收集的原始素材（**不進 Git**）
- `exports/`: 匯出檔（PDF/講義輸出/題庫匯出等，**不進 Git**）
- `exports/<date>_vN/manifest.md`: 匯出版次紀錄（可進 Git，二進位輸出檔仍忽略）
- `scripts/`: 維護腳本（結構稽核、後續自動化）
- `web/`: 互動式教學平台（React + TypeScript + Tailwind CSS v4 + shadcn/ui + react-router-dom）

## 工作原則

- 以 **病生理學 → 決策節點（decision points）→ 藥物選擇/劑量/路徑 → 風險控管** 的順序編排
- 劑量與流程以「台北市/所屬 EMS 系統現行 protocol + medical director 指示」為準；本 repo 僅作教學設計與知識框架
